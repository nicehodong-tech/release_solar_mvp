# 오행 핵심어 사전 v1

오행은 사주명리의 가장 기본적인 성정 언어다.  
이 사전은 오행 하나를 최종 통변으로 쓰기 위한 자료가 아니라, 오행 배합·십신·월령·격국 사전에서 다시 조합하기 위한 핵심어 자산이다.

## 기본 생극

- 상생: 목생화 → 화생토 → 토생금 → 금생수 → 수생목
- 상극: 목극토 → 토극수 → 수극화 → 화극금 → 금극목

## 사용 원칙

- 오행 하나로 성격·직업·재물·결혼을 확정하지 않는다.
- 오행은 기질의 방향을 보여준다.
- 월령, 조후, 강약, 천간 노출, 지지 기반, 십신으로 쓰임을 다시 판단한다.
- 아래 핵심어는 고객 문장으로 바로 복사하는 문장이 아니라 통변 재료다.

# 목(木)

**한 줄 핵심:** 목은 시작하고 자라며, 자기 가능성을 밖으로 뻗어내는 기운이다.

**중심 물상:** 새싹, 나무, 줄기, 성장, 봄, 방향성, 확장

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 성장 욕구 |
| 기질 | 진취성 |
| 기질 | 자기 방향 |
| 재능 | 기획력 |
| 재능 | 교육 감각 |
| 재능 | 확장 능력 |
| 관계 | 경쟁 의식 |
| 관계 | 동료 의식 |
| 직업 | 개척형 업무 |
| 직업 | 교육·기획 |
| 재물 | 성장 투자 |
| 리스크 | 끝맺음 부족 |

## 기초 성정

- 앞으로 나아가려는 마음
- 새로운 일을 시작하려는 기질
- 자기 방향을 세우려는 태도
- 지체되는 상황을 답답해함
- 비교와 경쟁에 반응함
- 자기 가능성을 시험하고 싶어 함

## 욕망

- 성장하고 싶은 욕망
- 인정받을 만한 발전을 만들고 싶은 마음
- 남보다 뒤처지지 않으려는 의식
- 자기 길을 열고 싶은 욕구

## 재능

- 새로운 기획
- 교육과 성장 관리
- 사람을 키우는 일
- 초기 시장 개척
- 아이디어를 방향으로 만드는 능력

## 관계

- 동료와 경쟁이 함께 생김
- 사람을 키우거나 이끌려 함
- 상대가 자신을 막는다고 느끼면 반발함
- 관계에서도 발전 가능성을 봄

## 직업

- 교육
- 기획
- 스타트업
- 브랜드 초기 구축
- 상담·코칭
- 조직 성장 관리

## 재물

- 성장 가능성에 돈을 씀
- 장기 투자와 확장에 관심
- 초기 비용과 미래 수익을 연결하려 함
- 결과가 늦으면 자금이 묶일 수 있음

## 리스크

- 시작만 많고 마무리가 약함
- 성장 욕심으로 무리함
- 경쟁심 때문에 방향을 자주 바꿈
- 가능성만 보고 현실 계산을 늦춤

## 과다·부족·청탁

| 상태 | 핵심어 |
|---|---|
| 과다 | 고집, 과도한 경쟁, 확장 과잉, 끝없는 시작, 조급함, 남과 비교 |
| 부족 | 무기력, 방향 상실, 성장 정체, 결정 지연, 자기주장 약화, 기회 회피 |
| 맑게 쓰임 | 건강한 성장, 교육성, 기획력, 자기 방향, 도전, 생명력 |
| 탁하게 쓰임 | 고집, 성급한 확장, 시기심, 미완성, 무리한 계획, 경쟁 피로 |

# 화(火)

**한 줄 핵심:** 화는 드러내고 비추며, 자기 존재를 사람들 앞에서 확인하려는 기운이다.

**중심 물상:** 태양, 불빛, 열, 여름, 시선, 표현, 밝음

|---|---|

| 관계 | 감정 과열 |
| 직업 | 대중 노출 |
| 직업 | 홍보·콘텐츠 |
| 재물 | 평판 수입 |
| 리스크 | 체면 손상 |

## 기초 성정

- 밖으로 표현하려는 마음
- 사람들 앞에서 반응을 확인하려는 기질
- 감정과 판단이 빠르게 드러남
- 침묵과 무관심을 답답해함
- 시선과 평판에 민감함
- 분위기를 띄우거나 이끌려 함

## 욕망

- 인정받고 싶은 욕망
- 보이고 싶은 마음
- 특별한 존재로 남고 싶은 의식
- 사람들 사이에서 존재감을 확인하려는 욕구

## 재능

- 말과 발표
- 분위기 조성
- 브랜딩과 이미지
- 교육과 설득
- 공개 무대에서의 반응성

## 관계

- 사람들의 반응을 빠르게 읽음
- 칭찬과 관심에 힘을 얻음
- 관계에서 감정 표현이 중요함
- 무시받거나 외면당하면 크게 흔들림

## 직업

- 강의
- 홍보
- 마케팅
- 방송·콘텐츠
- 공연
- 정치·대외 활동
- 교육

## 재물

- 인지도와 평판이 돈으로 연결됨
- 대중 반응이 수입에 영향을 줌
- 이미지와 브랜드에 돈을 씀
- 체면과 과시 비용이 커질 수 있음

## 리스크

- 감정 과열
- 말실수
- 평판 손상
- 허영과 과시
- 열정 뒤의 피로
- 주목 욕구로 인한 무리

|---|---|

| 부족 | 무표정, 표현 부족, 자신감 저하, 인지도 약화, 소극성, 의욕 저하 |

# 토(土)

**한 줄 핵심:** 토는 받아들이고 묶어두며, 생활과 책임을 현실의 형태로 만드는 기운이다.

**중심 물상:** 땅, 산, 밭, 창고, 중심, 생활, 기반

| 분류 | 핵심어 |
|---|---|
| 기질 | 현실 감각 |
| 기질 | 버티는 힘 |
| 기질 | 소유 의식 |
| 재능 | 관리 능력 |
| 재능 | 중재 능력 |
| 재능 | 축적 감각 |
| 관계 | 책임 의식 |
| 관계 | 가족 중심 |
| 직업 | 운영·관리 |
| 직업 | 부동산·자산 |
| 재물 | 자산 축적 |
| 리스크 | 고집과 정체 |

## 기초 성정

- 쉽게 흔들리지 않으려는 마음
- 생활을 유지하고 책임지려는 기질
- 확실한 기반과 자기 몫을 중요하게 봄
- 낯선 변화보다 안정된 구조를 선호함
- 사람과 돈을 받아내는 힘이 있음
- 느려 보여도 오래 버팀

## 욕망

- 안정된 생활을 만들고 싶은 욕망
- 자기 땅과 자기 사람을 지키고 싶은 마음
- 가족과 자산을 책임지고 싶은 의식
- 흔들리지 않는 중심이 되고 싶은 욕구

## 재능

- 운영 관리
- 중재
- 부동산·자산 감각
- 장기 계획
- 생활 기반 만들기

## 관계

- 가까운 사람을 책임지려 함
- 관계가 생활 문제로 이어짐
- 정이 들면 쉽게 끊지 못함
- 상대가 책임을 피하면 강하게 실망함

## 직업

- 운영
- 관리
- 행정
- 부동산
- 건설
- 자산 관리
- 중간 책임자

## 재물

- 자산으로 남기는 힘
- 주거와 부동산 관심
- 장기 축적
- 가족과 생활비 지출
- 현금이 자산에 묶일 수 있음

## 리스크

- 고집
- 정체
- 미련
- 책임 과다

# 금(金)

**한 줄 핵심:** 금은 가르고 다듬고, 무엇이 맞고 틀린지 기준을 세우는 기운이다.

**중심 물상:** 칼, 보석, 금속, 가을, 절단, 평가, 완성

| 분류 | 핵심어 |
|---|---|
| 기질 | 분별력 |
| 기질 | 자존심 |
| 기질 | 냉정한 판단 |

| 재능 | 품질 관리 |
| 재능 | 결단력 |
| 관계 | 선 긋기 |
| 관계 | 평가 의식 |
| 직업 | 심사·검수 |
| 직업 | 법·기술 |
| 재물 | 가치 선별 |
| 리스크 | 차가운 단절 |

## 기초 성정

- 옳고 그름을 가르려는 마음
- 완성도와 기준을 중요하게 봄
- 사람과 물건의 가치를 빠르게 판단함
- 흐릿한 관계와 애매한 약속을 싫어함
- 말과 태도가 날카롭게 보일 수 있음
- 자존심과 수치심이 강하게 작용함

## 욕망

- 정확한 평가를 받고 싶은 욕망
- 함부로 취급받지 않으려는 마음
- 실력과 품질로 인정받고 싶은 의식
- 불필요한 것을 잘라내고 싶은 욕구

## 재능

- 검수
- 심사
- 정밀 기술
- 법과 규정
- 디자인 완성도
- 비용 절감과 품질 관리

## 관계

- 사람을 쉽게 들이지 않음
- 무례와 허술함에 민감함
- 실망하면 차갑게 선을 긋는 편
- 존중과 수준을 중요하게 봄

## 직업

- 법률
- 기술
- 의료·시술
- 품질 관리
- 금융 심사
- 편집·검수
- 디자인 완성

## 재물

- 가치 있는 것을 고르는 감각
- 품질과 희소성에 돈이 붙음
- 낭비를 줄이고 구조를 다듬음
- 고급 취향과 체면 비용이 생길 수 있음

## 리스크

- 비판 과다
- 냉정한 단절
- 완벽주의

**한 줄 핵심:** 수는 기억하고 흘러가며, 보이지 않는 정보와 감정을 저장하는 기운이다.
