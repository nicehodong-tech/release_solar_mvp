# 10천간 핵심어 사전 v1

10천간은 오행보다 한 단계 더 구체적인 통변 단위다.  
같은 목이라도 甲과 乙은 다르고, 같은 화라도 丙과 丁은 다르다.  
이 사전은 천간이 드러내는 의식, 표현, 사회적 태도, 욕망, 재능의 방향을 핵심어 중심으로 정리한 자료다.

## 사용 원칙

- 천간 하나로 인생 전체를 확정하지 않는다.
- 천간은 밖으로 드러나는 의식과 사회적 표현을 강하게 보여준다.
- 실제 현실성은 지지 기반, 지장간, 투출·통근, 월령과 조후, 십신 배합으로 다시 본다.
- 아래 핵심어는 고객 문장 자체가 아니라 통변 재료다.

## 10천간 구분

# 甲 갑목

**오행·음양:** 목 · 양  
**한 줄 핵심:** 갑목은 큰 방향을 세우고, 자기 길을 곧게 밀고 나가려는 천간이다.

**중심 물상:** 큰 나무, 기둥, 대들보, 숲의 중심, 위로 뻗는 줄기.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 직선성 |
| 기질 | 자존심 |
| 기질 | 독립심 |
| 재능 | 기획력 |
| 재능 | 리더십 |
| 재능 | 개척력 |
| 관계 | 동등함 요구 |
| 관계 | 보호 본능 |
| 직업 | 교육·기획 |
| 직업 | 조직 개척 |
| 재물 | 성장 투자 |
| 리스크 | 고집과 독선 |

## 기초 성정

- 자기 방향이 분명하다
- 남에게 쉽게 끌려가지 않는다
- 큰 목표와 명분을 중요하게 본다
- 정면으로 부딪히는 성향이 있다
- 답답한 환경을 오래 견디기 어렵다

## 욕망

- 자기 이름으로 서고 싶은 욕망
- 큰 인물로 인정받고 싶은 마음
- 남에게 꺾이지 않으려는 자존심
- 작은 실속보다 큰 방향을 잡고 싶은 욕구

## 재능

- 기획
- 교육
- 조직 개척
- 장기 프로젝트
- 사람을 이끄는 힘
- 큰 방향을 제시하는 능력

## 관계

- 동등한 관계를 원한다
- 상대가 자신을 억누르면 강하게 반발한다
- 가까운 사람을 보호하려는 마음이 있다
- 존중받지 못하면 관계를 쉽게 낮춰 본다

## 직업

- 교육
- 기획
- 경영
- 공공 리더
- 브랜드 초기 구축
- 조직 개척
- 컨설팅

## 재물

- 성장 가능성에 돈을 쓴다
- 초기 투자와 확장 비용이 커질 수 있다
- 큰 계획에는 강하지만 작은 정산을 놓치기 쉽다
- 자기 이름의 수입을 원한다

## 연애·결혼

- 사랑에서도 존중과 동등함을 본다
- 상대를 이끌려 하지만 간섭받는 것은 싫어한다
- 자존심 싸움이 생기기 쉽다
- 믿으면 오래 책임지려 한다

## 리스크

- 고집
- 독선
- 타협 부족
- 큰소리와 현실의 불일치
- 손절 지연
- 자존심 손상

## 청탁·과다·부족

| 상태 | 핵심어 |
|---|---|
| 맑게 쓰임 | 큰 방향, 성장성, 리더십, 교육성, 개척력, 주체성 |
| 탁하게 쓰임 | 독선, 명분 과잉, 체면 고집, 강압, 현실 계산 부족, 경직 |
| 과다 | 자기 뜻만 밀어붙임, 남의 조언을 간섭으로 봄, 손실을 인정하지 않음, 큰 목표에 취해 세부를 놓침 |
| 부족 | 방향 상실, 자기주장 약화, 결단 부족, 큰 목표 부재, 남에게 끌려감 |

## 보정 포인트

- 월령·조후: 월령을 얻으면 큰 기세와 추진력이 살아난다. 실령하거나 한습하면 이상은 있으나 출발이 늦고, 금이 강하면 꺾임과 교정이 강해진다.
- 지지 기반: 지지에 寅卯亥未 등 목의 기반이 있으면 실제 행동력과 지속성이 강해진다. 무근하면 말과 계획이 앞서기 쉽다.
- 지장간·투출: 지장간의 甲은 겉으로는 조용해도 속의 자존심과 독립심으로 남는다. 운에서 투출하면 그동안 눌린 자기 방향이 밖으로 나온다.

# 乙 을목

**오행·음양:** 목 · 음  
**한 줄 핵심:** 을목은 유연하게 파고들고, 관계 속에서 살아남는 천간이다.

**중심 물상:** 풀, 덩굴, 꽃, 작은 가지, 바람에 휘어도 다시 살아나는 생명력.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 유연성 |
| 기질 | 눈치 |
| 기질 | 생존력 |
| 재능 | 관계 감각 |
| 재능 | 미감 |
| 재능 | 적응력 |
| 관계 | 호감 관리 |
| 관계 | 선택적 의존 |
| 직업 | 상담·디자인 |
| 직업 | 서비스·기획 |
| 재물 | 관계 수익 |
| 리스크 | 우회와 질투 |

## 기초 성정

- 사람과 분위기를 빠르게 읽는다
- 정면 충돌보다 우회와 조율을 선호한다
- 부드럽지만 자기 욕심이 분명하다
- 환경에 맞춰 살아남는 힘이 있다
- 섬세한 감각과 취향이 있다

## 욕망

- 사랑받고 선택받고 싶은 욕망
- 자기 자리를 잃지 않으려는 마음
- 부드럽게 이기고 싶은 의식
- 관계 안에서 살아남고 확장하려는 욕구

## 재능

- 관계 조율
- 디자인과 미감
- 상담
- 서비스 기획
- 환경 적응
- 작은 틈을 이용하는 능력

## 관계

- 호감을 잘 얻는다
- 상대에게 맞추면서도 속으로는 계산한다
- 비교와 질투가 생기기 쉽다
- 정면으로 싸우기보다 관계를 움직인다

## 직업

- 상담
- 디자인
- 미용
- 패션
- 교육
- 서비스
- 브랜딩
- 고객 관리

## 재물

- 관계와 취향을 통해 돈을 번다
- 고객 반응을 읽는 감각이 있다
- 사람에게 쓰는 비용이 커질 수 있다
- 가격을 강하게 부르기보다 분위기를 본다

## 연애·결혼

- 연애에서 호감과 분위기를 중요하게 본다
- 상대의 태도 변화에 예민하다
- 사랑받고 싶지만 만만해 보이는 것은 싫어한다
- 미련과 질투가 함께 생기기 쉽다

## 리스크

- 질투
- 비교
- 우회적 공격
- 결정 지연
- 관계 피로
- 자기 기준 숨김

## 청탁·과다·부족

| 상태 | 핵심어 |
|---|---|
| 맑게 쓰임 | 유연성, 생존력, 감각, 관계성, 섬세함, 조율 |
| 탁하게 쓰임 | 질투, 의존, 계산적 태도, 우회, 변덕, 관계 혼잡 |
| 과다 | 상대 반응에 지나치게 흔들림, 직접 말하지 않고 돌려서 움직임, 질투와 비교가 심해짐, 좋은 사람으로 보이려다 자기 몫을 놓침 |
| 부족 | 관계 감각 약화, 적응력 부족, 미감 약화, 호감 형성 어려움, 생존의 유연성 부족 |

## 보정 포인트

- 월령·조후: 봄과 여름에는 생장과 표현이 살아난다. 가을 금기가 강하면 평가와 상처가 많아지고, 겨울 수기가 많으면 생각과 감정은 깊지만 현실화가 늦어진다.
- 지지 기반: 卯를 만나면 순수한 을목성이 강하고, 亥未와 연결되면 관계·감정·성장성이 커진다. 酉金이 강하면 미감과 평가가 함께 날카로워진다.
- 지장간·투출: 지장간의 乙은 숨은 관계 욕구와 미련으로 남는다. 투출하면 사람을 의식하고 조율하는 모습이 뚜렷해진다.

# 丙 병화

**오행·음양:** 화 · 양  
**한 줄 핵심:** 병화는 세상을 비추고, 자기 존재를 공개적으로 드러내는 천간이다.

**중심 물상:** 태양, 대낮, 큰 불빛, 공개된 무대, 만물을 비추는 시선.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 공개성 |
| 기질 | 자신감 |
| 기질 | 명료함 |
| 재능 | 리더십 |
| 재능 | 설득력 |
| 재능 | 대중성 |
| 관계 | 인정 욕구 |
| 관계 | 밝은 존재감 |
| 직업 | 교육·홍보 |
| 직업 | 공적 활동 |
| 재물 | 평판 수익 |
| 리스크 | 과시와 체면 |

## 기초 성정

- 숨기기보다 드러내려 한다
- 말과 표정이 비교적 분명하다
- 사람들 앞에서 존재감이 생긴다
- 옳고 그름을 규정하려는 성향이 있다
- 어두운 분위기를 답답해한다

## 욕망

- 공개적으로 인정받고 싶은 욕망
- 중심에 서고 싶은 마음
- 자기 관점을 사람들과 공유하려는 의식
- 밝고 큰 사람으로 보이고 싶은 욕구

## 재능

- 강의
- 발표
- 지도력
- 홍보
- 대중 설득
- 공적 커뮤니케이션

## 관계

- 사람들을 모으고 방향을 제시한다
- 관심과 칭찬에 힘을 얻는다
- 무시당하면 자존심이 크게 상한다
- 관계에서 솔직함과 반응을 원한다

## 직업

- 교육
- 홍보
- 정치
- 방송
- 공공 활동
- 마케팅
- 리더 역할

## 재물

- 평판과 인지도가 돈이 된다
- 공개된 성과가 수입으로 이어진다
- 브랜드와 이름값을 키우려 한다
- 체면 비용과 과시 지출이 생길 수 있다

## 연애·결혼

- 연애에서 적극적이고 표현이 비교적 분명하다
- 상대가 반응해주기를 원한다
- 사랑받는다는 확인이 필요하다
- 자존심 때문에 사과가 늦을 수 있다

## 리스크

- 과시
- 말실수
- 체면 손상
- 감정 과열
- 자기중심성
- 평판 불안

## 청탁·과다·부족

| 상태 | 핵심어 |
|---|---|
| 맑게 쓰임 | 명료함, 밝음, 지도력, 대중성, 설득력, 인지도 |
| 탁하게 쓰임 | 허세, 자기과시, 말의 과장, 체면 집착, 감정 과열, 타인 압박 |
| 과다 | 자기 관점만 정답으로 여김, 사람들의 반응에 지나치게 의존, 드러나는 것만 중시, 평판 때문에 무리함 |
| 부족 | 자신감 저하, 표현 부족, 존재감 약화, 사회적 방향감 부족, 인정 욕구가 숨어서 불안으로 남음 |

## 보정 포인트

- 월령·조후: 여름에는 병화의 기세가 강해 과열과 과시를 조심해야 한다. 겨울에는 병화가 조후의 핵심이 되어 사람을 살리고 방향을 제시하는 힘이 된다.
- 지지 기반: 巳午寅을 만나면 공개성과 추진력이 강해진다. 申子辰 수기가 강하면 병화가 불안해지고, 壬癸와의 관계에서 지도력과 통제 욕구가 달라진다.
- 지장간·투출: 지장간의 丙은 숨은 자존심과 드러나고 싶은 욕망이다. 투출하면 말, 표정, 사회적 활동으로 명확해진다.

# 丁 정화

**오행·음양:** 화 · 음  
**한 줄 핵심:** 정화는 한곳에 집중해 어둠을 밝히고, 섬세한 열로 의미를 만드는 천간이다.

**중심 물상:** 등불, 촛불, 별빛, 작업대의 불, 안쪽에서 오래 타는 열.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 섬세함 |
| 기질 | 집중력 |
| 기질 | 감수성 |
| 재능 | 창작력 |
| 재능 | 상담 감각 |
| 재능 | 정밀 표현 |
| 관계 | 정서 반응 |
| 관계 | 특별한 애정 |
| 직업 | 예술·상담 |
| 직업 | 교육·기술 |
| 재물 | 감성 수익 |
| 리스크 | 불안과 집착 |

## 기초 성정

- 겉으로 요란하지 않아도 안쪽의 열이 강하다
- 사람의 감정과 분위기를 세밀하게 읽는다
- 한 가지에 오래 집중하는 힘이 있다
- 무심한 환경에서 쉽게 상처받는다
- 작고 정확한 표현을 중요하게 본다

## 욕망

- 깊이 이해받고 싶은 욕망
- 특별한 사람으로 인정받고 싶은 마음
- 자기만의 감각을 표현하고 싶은 의식
- 차갑고 무심한 세계를 밝히고 싶은 욕구

## 재능

- 예술
- 글
- 디자인
- 상담
- 교육
- 정밀 기술
- 섬세한 표현

## 관계

- 가까운 관계에서 깊어진다
- 특별한 대우와 정서적 반응을 원한다
- 무심한 말투에 상처받기 쉽다
- 사랑이 깊어지면 집착이 생길 수 있다

## 직업

- 예술
- 상담
- 교육
- 심리
- 디자인
- 공예
- 콘텐츠
- 정밀 기술

## 재물

- 감성, 취향, 기술이 돈이 된다

- 작은 세계 안에서 확실한 의미를 얻고 싶은 의식
