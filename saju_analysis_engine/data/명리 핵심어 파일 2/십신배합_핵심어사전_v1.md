# 십신 배합 핵심어 사전 v1

십신 배합은 십신 하나보다 실제 통변에 가깝다.  
두 십신이 만나면 성격이 아니라 **작용 상태, 생활 사건, 직업성, 재물 방식, 관계 문제**가 만들어진다.

## 사용 원칙

- 두 십신을 단순히 더하지 않는다.
- 생하는가, 극하는가, 다투는가, 제어하는가, 혼잡한가를 먼저 본다.
- 같은 배합도 월령, 천간 노출, 지지 기반, 투출·통근, 강약, 조후에 따라 결과가 달라진다.
- 이 사전은 고객 문장 자체가 아니라 통변과 섹션 선택을 위한 핵심어 자산이다.

## 수록 범위

10개 십신의 2글자 조합 45개를 수록했다.

# 비견+겁재

**분류:** 비겁+비겁  
**작용:** 동류·경쟁  
**한 줄 핵심:** 비견과 겁재는 자기 기준이 강해지고, 사람 속에서 몫과 주도권을 다투는 배합이다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 독립성 |
| 기질 | 승부욕 |
| 기질 | 자기주장 |
| 재능 | 무리 장악 |
| 재능 | 생존력 |
| 재능 | 동료 활용 |
| 관계 | 친구·형제 문제 |
| 관계 | 동업 긴장 |
| 직업 | 대중 상대 |
| 직업 | 경쟁 시장 |
| 재물 | 몫 다툼 |
| 리스크 | 군겁쟁재 |

## 중심 작용

- 내 힘과 사람의 힘이 동시에 커진다.
- 혼자 서려는 마음과 무리 안에서 이기려는 마음이 같이 작용한다.

## 기질

- 자기주장이 강하고 쉽게 꺾이지 않는다.
- 사람이 많아질수록 존재감이 커진다.
- 경쟁이 있으면 힘이 난다.

## 관계

- 친구·형제·동료가 기회와 갈등을 동시에 만든다.
- 같은 편 안에서도 주도권 다툼이 생긴다.

## 직업

- 영업, 조직 관리, 대중 상대 사업, 스포츠, 경쟁 시장, 프랜차이즈형 분야

## 재물

- 수입보다 분배가 문제다.
- 공동 수익, 동업, 친구·형제와의 돈 문제가 커질 수 있다.

## 연애·결혼

- 연애에서는 주변 사람, 경쟁자, 자존심 문제가 끼기 쉽다.

## 리스크

- 동업 갈등
- 손재
- 친구·형제 문제
- 주도권 싸움
- 분배 기준 붕괴

## 좋아지는 조건

- 사람을 모아 생산이나 판매로 연결할 때
- 관성이나 계약 기준이 분명할 때

## 나빠지는 조건

- 사람은 많지만 정산 기준이 없을 때
- 재성을 직접 다툴 때

## 관련 통변어

- 군겁쟁재
- 비견쟁재
- 동업
- 분배 갈등
- 대중성

# 비견+식신

**분류:** 비겁+식상  
**작용:** 생산  
**한 줄 핵심:** 비견과 식신은 자기 힘으로 기술과 결과물을 꾸준히 만들어내는 배합이다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 자기 페이스 |
| 기질 | 꾸준함 |
| 기질 | 독립 생산 |
| 재능 | 기술 숙련 |
| 재능 | 콘텐츠 생산 |
| 재능 | 교육 능력 |
| 관계 | 편한 관계 |
| 관계 | 동료 생산성 |
| 직업 | 기술·교육 |
| 직업 | 프리랜스 |
| 재물 | 결과물 수입 |
| 리스크 | 판매 지연 |

## 중심 작용

- 내 힘이 결과물로 빠져나간다.
- 자기 기술과 자기 페이스가 중요하다.

## 기질

- 느리더라도 자기 방식으로 만든다.
- 남의 간섭이 적을수록 생산성이 산다.

## 관계

- 편한 사람들과 오래 일한다.
- 관계보다 결과물을 중심으로 연결된다.

## 직업

- 교육, 기술, 요식, 콘텐츠, 강의, 프리랜스 제작, 제품 생산

## 재물

- 만든 것으로 돈을 벌 수 있다.
- 재성이 없으면 생산은 되지만 판매와 가격 책정이 늦다.

## 연애·결혼

- 연애에서는 편안함, 생활감, 안정된 애정 표현이 강하다.

## 리스크

- 안주
- 느린 수익화
- 취미화
- 자기 페이스 고집
- 판매 구조 부족

## 좋아지는 조건

- 기술에 가격이 붙을 때
- 동료와 생산 시스템이 생길 때

## 나빠지는 조건

- 계속 만들기만 하고 시장에 내놓지 않을 때

## 관련 통변어

- 비견생식
- 식신생재 전 단계
- 기술형 독립
- 콘텐츠 생산

# 비견+상관

**분류:** 비겁+식상  
**작용:** 표현·반항  
**한 줄 핵심:** 비견과 상관은 자기 생각을 직접 드러내고 기존 기준에 맞서려는 배합이다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 자기표현 |
| 기질 | 반항심 |
| 기질 | 독립성 |
| 재능 | 언변 |
| 재능 | 콘텐츠 감각 |
| 재능 | 비판력 |
| 관계 | 말의 충돌 |
| 관계 | 간섭 거부 |
| 직업 | 프리랜스·콘텐츠 |
| 직업 | 개선·기획 |
| 재물 | 재능 수익 |
| 리스크 | 구설 |

## 중심 작용

- 자기 힘이 말과 표현으로 강하게 나온다.
- 누가 시키는 방식보다 자기 방식으로 말하고 만들려 한다.

## 기질

- 말이 곧 자존심이다.
- 틀린 것을 보고 참기 어렵다.
- 간섭을 받으면 바로 반응한다.

## 관계

- 관계에서 말이 세고 선이 분명하다.
- 상대가 통제하려 들면 부딪힌다.

## 직업

- 콘텐츠, 방송, 디자인, 마케팅, 기술 개선, 프리랜스, 비평·기획

## 재물

- 재능과 표현이 돈이 된다.
- 계약과 평판 관리가 약하면 손실도 빠르다.

## 연애·결혼

- 연애에서는 매력과 자극이 있지만 말로 다투기 쉽다.

## 리스크

- 말실수
- 상사와 충돌
- 평판 손상
- 규칙 위반
- 독선적 표현

## 좋아지는 조건

- 표현이 상품과 실적으로 연결될 때

## 나빠지는 조건

- 자기 말만 옳다고 밀어붙일 때

## 관련 통변어

- 상관기질
- 독립 콘텐츠
- 상관생재 후보
- 상사 충돌

# 비견+정재

**분류:** 비겁+재성  
**작용:** 소유·쟁재  
**한 줄 핵심:** 비견과 정재는 자기 몫과 안정된 돈이 맞물려 소유와 정산을 강하게 의식하는 배합이다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 소유 의식 |
| 기질 | 자기 몫 |
| 기질 | 현실성 |
| 재능 | 생활 관리 |
| 재능 | 정산 감각 |
| 재능 | 독립 수입 |
| 관계 | 돈의 선 |
| 관계 | 배우자 조건 |
| 직업 | 관리·회계 |
| 직업 | 개인 사업 |
| 재물 | 고정 수입 |
| 리스크 | 쟁재 |

## 중심 작용

- 내 돈, 내 몫, 내 생활 기준이 강해진다.
- 재물을 안정적으로 갖고 싶지만 주변과 다툴 수 있다.

## 기질

- 현실적이고 계산이 분명하다.
- 자기가 번 돈을 지키려 한다.

## 관계

- 가까운 관계에서도 돈의 선이 중요하다.
- 동료·배우자·가족과 정산 문제가 생기기 쉽다.

## 직업

- 회계, 행정, 개인 사업, 관리, 안정적 실무, 월급 기반 직업

## 재물

- 고정 수입을 원한다.
- 비겁이 재성을 건드리면 돈이 사람과 나뉜다.

## 연애·결혼

- 결혼에서는 경제 기준, 생활비, 소유 문제가 중요하다.

## 리스크

- 쟁재
- 배우자와 돈 문제
- 동업 정산
- 가족 지출
- 생활비 갈등

## 좋아지는 조건

- 돈과 역할이 분명할 때
- 관성으로 계약과 책임이 정리될 때

## 나빠지는 조건

- 가까운 사람과 돈을 말로만 정할 때

## 관련 통변어

- 비견쟁재
- 정재 손상
- 생활비 갈등
- 배우자 조건

# 비견+편재

**분류:** 비겁+재성  
**작용:** 쟁재·사업성  
**한 줄 핵심:** 비견과 편재는 큰돈과 기회를 자기 손으로 잡으려 하지만 사람과 분배 문제가 생기는 배합이다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 독립 욕심 |
| 기질 | 큰돈 감각 |
| 기질 | 승부욕 |
| 재능 | 사업성 |
| 재능 | 시장 감각 |
| 재능 | 직접 영업 |
| 관계 | 동업 긴장 |
| 관계 | 이성·돈 혼잡 |
| 직업 | 사업·영업 |
| 직업 | 거래·투자 |
| 재물 | 큰돈의 이동 |
| 리스크 | 손재 |

## 중심 작용

- 자기 힘으로 큰돈을 잡으려 한다.
- 사람과 기회가 많지만 돈의 분배가 핵심 문제가 된다.

## 기질

- 기회를 보면 직접 움직인다.
- 남 밑에서 작은 돈만 받는 것을 답답해한다.

## 관계

- 사람을 통해 돈이 들어오고, 사람 때문에 돈이 나간다.

## 직업

- 사업, 영업, 무역, 유통, 투자, 부동산 거래, 프리랜스 영업

## 재물

- 큰돈의 흐름을 만들 수 있다.
- 쟁재가 강하면 들어와도 남기기 어렵다.

## 연애·결혼

- 연애에서는 외부 인연, 자유, 돈과 관계의 혼잡이 생기기 쉽다.

## 리스크

- 손재
- 동업 갈등
- 외부 이성 문제
- 투기
- 분배 싸움

## 좋아지는 조건

- 식상이 재성을 생하고 관성이 기준을 잡을 때

## 나빠지는 조건

- 사람과 돈을 기준 없이 섞을 때

## 관련 통변어

- 군겁쟁재
- 편재사업
- 동업 손재
- 외부 인연

# 비견+정관

**분류:** 비겁+관성  
**작용:** 자기 기준과 질서  
**한 줄 핵심:** 비견과 정관은 자기 기준을 사회적 책임과 공식 질서 안에 세우려는 배합이다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 책임 의식 |
| 기질 | 자기 원칙 |
| 기질 | 체면 |
| 재능 | 조직 내 독립성 |
| 재능 | 역할 수행 |
| 재능 | 신뢰 구축 |
| 관계 | 동등한 존중 |
| 관계 | 공식 관계 |
| 직업 | 관리·조직 |
| 직업 | 공공·행정 |
| 재물 | 직위 수입 |
| 리스크 | 권위 충돌 |

## 중심 작용

- 자기 기준이 있지만 사회적 규칙을 무시하지 않는다.
- 조직 안에서 자기 역할과 이름을 세우려 한다.

## 기질

- 책임감과 독립심이 같이 있다.
- 시키는 대로만 하기보다 자기 기준으로 잘하려 한다.

## 관계

- 관계에서 예의와 동등한 존중을 요구한다.

## 직업

- 공무원, 대기업, 관리직, 행정, 법무, 교직, 조직 책임자

## 재물

- 직위와 역할이 수입의 기반이 된다.

## 연애·결혼

- 결혼에서는 책임 있는 관계를 원하지만 동등함도 강하게 요구한다.

## 리스크

- 권위와 자존심 충돌
- 상사와 기준 차이
- 체면 부담
- 융통성 부족

## 좋아지는 조건

- 공식 권한과 자기 역할이 함께 있을 때

## 나빠지는 조건

- 책임은 있는데 자율성이 없을 때

## 관련 통변어

- 비견대관
- 조직 속 독립성
- 공식 역할
- 권위 충돌

# 비견+편관

**분류:** 비겁+관성  
**작용:** 실전 경쟁  
**한 줄 핵심:** 비견과 편관은 자기 힘으로 압박과 경쟁을 정면에서 버티는 배합이다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 강한 자존심 |
| 기질 | 위기 반응 |
| 기질 | 승부욕 |
| 재능 | 실전력 |
| 재능 | 돌파력 |
| 재능 | 위기 대응 |
| 관계 | 강한 상대 |
| 관계 | 권력 긴장 |
| 직업 | 군경·위기 |
| 직업 | 현장 책임 |
| 재물 | 고위험 수익 |
| 리스크 | 강한 충돌 |

## 중심 작용

- 나와 압박이 정면으로 만난다.
- 위기와 경쟁을 피하기보다 버티며 증명하려 한다.

## 기질

- 쉽게 겁먹지 않는다.
- 강한 상대 앞에서 더 강해진다.

## 관계

- 관계에서 주도권과 힘의 균형을 크게 본다.

## 직업

- 군경, 소방, 감사, 위험 산업, 현장 관리, 스포츠, 고압 영업

## 재물

- 위험과 성과가 큰 자리에서 돈이 붙을 수 있다.

## 연애·결혼

- 연애에서는 강한 상대에게 끌리지만 주도권 싸움이 생기기 쉽다.

## 리스크

- 사고성
- 권력 다툼
- 폭발적 분노
- 압박 과다
- 관계 손상

## 좋아지는 조건

- 식신이나 인성이 편관을 다룰 때

## 나빠지는 조건

- 힘으로만 버티고 조율이 없을 때

## 관련 통변어

- 비견대살
- 실전형 기질
- 권력 긴장
