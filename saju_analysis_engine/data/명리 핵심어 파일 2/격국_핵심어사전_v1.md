# 격국 핵심어 사전 v1

격국은 십신의 성격을 다시 설명하는 항목이 아니다.  
월령에서 시작된 기능이 사회에서 어떤 역할과 쓰임을 얻는지, 그 역할을 성립시키는 조건과 손상·복구 관계를 보는 기준이다.

## 사용 원칙

- 격은 월령을 출발점으로 정한다.
- 월지 지장간의 사령, 투출, 통근, 실제 세력을 함께 본다.
- 격의 이름만으로 직업, 부, 명예, 결혼을 확정하지 않는다.
- 상신은 격을 성립시키는 조건과 능력이다.
- 기신은 격이나 상신을 손상하는 요소다.
- 구신은 이미 생긴 손상을 복구하는 요소다.
- 상신·기신·구신은 십신 이름으로 고정하지 않고 실제 배합에 따라 정한다.
- 격국용신, 억부용신, 조후용신, 병약의 약, 통관 오행을 같은 뜻으로 사용하지 않는다.
- 이 사전은 고객 문장 자체가 아니라 통변과 섹션 선택을 위한 핵심어 자산이다.

## 격을 정하는 순서

1. 월령과 실제 사령을 확인한다.
2. 월지 지장간 가운데 무엇이 투출하고 통근했는지 본다.
3. 격이 월령의 힘을 실제로 쓰는지, 이름만 남았는지 구분한다.
4. 상신이 격을 돕는지, 기신이 격이나 상신을 손상하는지 본다.
5. 구신이 손상을 복구할 수 있는지 본다.
6. 청탁, 혼잡, 거리, 위치, 대운·세운의 변화를 확인한다.
7. 직업·재물·관계·권한의 현실 사건으로 넘긴다.

## 수록 범위

- 정격 8개: 정관격·편관격·정인격·편인격·식신격·상관격·정재격·편재격
- 비겁 월령의 특수 정격 2개: 건록격·양인격
- 외격: 종격·화격·전왕격은 보조 기준으로 별도 수록
- 각 정격: `기질 3 / 재능 3 / 관계 2 / 직업 2 / 재물 1 / 리스크 1`

## 격국 기본 용어

| 용어 | 핵심 의미 |
|---|---|
| 격 | 월령에서 출발한 사회적 역할과 쓰임 |
| 상신 | 격이 제 역할을 하게 만드는 핵심 조건 |
| 기신 | 격이나 상신의 기능을 손상하는 요소 |
| 구신 | 손상된 격이나 상신을 복구하는 요소 |
| 성격(成格) | 격과 상신이 실제 기능을 갖춘 상태 |
| 파격(破格) | 격의 중심 기능이나 상신이 손상된 상태 |
| 청 | 역할과 배합이 분명하고 혼잡이 적은 상태 |
| 탁 | 역할이 겹치거나 상충해 결과가 흐려진 상태 |

# 01. 정관격 (正官格)

**격의 중심:** 공식 기준 · 책임 · 직책 · 신뢰  
**한 줄 핵심:** 정관격은 사회가 정한 기준을 지키며 역할, 직책, 평판을 안정적으로 확보하는 격이다.  
**성립 핵심:** 정관이 월령의 힘을 얻고 맑게 쓰이며, 일간이 그 책임을 감당하고 재성·인성 등이 관의 기능을 이어줄 때 성립한다.  
**대표 성격 조합:** 재생관 · 관인상생  
**대표 손상:** 상관견관 · 관살혼잡 · 관다신약  
**상신 후보:** 재성의 생조 · 인성의 연결과 보호  
**기신 후보:** 정관을 직접 치는 상관 · 역할을 흐리는 편관 · 감당하기 어려운 관성 과다  
**구신 방향:** 인성으로 상관을 제어하거나, 혼잡한 관살의 역할을 정리하고, 약한 일간의 기반을 보완한다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 책임 의식 |
| 기질 | 기준 준수 |
| 기질 | 평판 관리 |
| 재능 | 규정 이해 |
| 재능 | 조직 적응 |
| 재능 | 공식 업무 처리 |
| 관계 | 예의와 경계 |
| 관계 | 권위와의 안정적 관계 |
| 직업 | 행정·공공·법무 |
| 직업 | 조직 관리·감사 |
| 재물 | 급여·직책 보상 |
| 리스크 | 평판 손상과 징계 |

## 현실화

- 조직에서 맡은 역할과 책임 범위가 분명해진다.
- 자격, 승진, 직책, 공식 계약처럼 제도 안의 성과로 이어진다.
- 신뢰를 오래 쌓아 권한을 얻는 방식에 강하다.

## 성립할 때

- 원칙을 실제 업무와 성과로 연결한다.
- 직책과 책임이 함께 올라가며 조직의 신뢰를 얻는다.

## 손상될 때

- 규정은 많지만 실제 권한은 약해진다.
- 말과 평판 문제로 상사, 기관, 계약 관계와 부딪힌다.

# 02. 편관격·칠살격 (偏官格·七殺格)

**격의 중심:** 압박 · 위험 · 결단 · 통제 · 권한  
**한 줄 핵심:** 편관격은 강한 경쟁과 위험을 제어해 위기 대응, 결단, 권한으로 바꾸는 격이다.  
**성립 핵심:** 편관이 월령의 힘을 얻되 일간을 무너뜨리지 않고, 식신의 제어 또는 인성의 전환처럼 다룰 방법이 있을 때 성립한다.  
**대표 성격 조합:** 식신제살 · 살인상생 · 양인가살  
**대표 손상:** 살중신경 · 재성으로 살이 과해짐 · 제화혼잡  
**상신 후보:** 식신의 제어 · 인성의 전환 · 양인의 보조  
**기신 후보:** 과한 편관을 다시 생하는 재성 · 무근한 일간 · 서로 충돌하는 식신과 인성  
**구신 방향:** 과한 살은 식신으로 제어하거나 인성으로 받아 쓰고, 약한 일간에는 먼저 감당할 기반을 마련한다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 긴장감 |
| 기질 | 결단력 |
| 기질 | 승부 감각 |
| 재능 | 위기 대응 |
| 재능 | 강한 통솔 |
| 재능 | 위험 통제 |
| 관계 | 위계와 긴장 |
| 관계 | 강한 사람과의 경쟁 |
| 직업 | 군경·보안·안전 |
| 직업 | 의료·기술·위기 관리 |
| 재물 | 성과·위험 부담 보상 |
| 리스크 | 사고·권력 충돌·법적 문제 |

## 현실화

- 남들이 피하는 문제와 책임을 맡아 해결한다.
- 위기 상황에서 결정을 내리고 사람과 자원을 통제한다.
- 제어가 되면 권한과 전문성이 되고, 제어가 없으면 압박과 위험이 된다.

## 성립할 때

- 강한 압박을 기술, 규정, 훈련으로 다룬다.
- 중책과 위험을 감당한 만큼 권한과 보상을 얻는다.

## 손상될 때

- 책임은 커지지만 판단과 보호 장치가 부족하다.
- 경쟁과 권한 다툼이 사고, 징계, 소송으로 커질 수 있다.

# 03. 정인격 (正印格)

**격의 중심:** 학습 · 자격 · 보호 · 문서 · 정통성  
**한 줄 핵심:** 정인격은 지식과 제도를 익혀 자격, 문서, 보호, 신뢰할 수 있는 전문성으로 쓰는 격이다.  
**성립 핵심:** 정인이 월령의 힘을 얻고 일간을 적절히 생하며, 관성이 인성을 생하거나 배운 것을 실제 역할로 쓸 통로가 있을 때 성립한다.  
**대표 성격 조합:** 관인상생 · 재관인 유통  
**대표 손상:** 재성괴인 · 인다무용 · 인성과다로 식상 억제  
**상신 후보:** 관성의 생조 · 인성이 지나칠 때 재성의 조건부 조절  
**기신 후보:** 필요한 인성을 치는 재성 · 결과를 막는 인성 과다 · 제도와 현실의 단절  
**구신 방향:** 재성이 필요한 인성을 손상하면 비겁이나 관성으로 보호하고, 인성이 지나치면 식상·재성으로 결과를 꺼낸다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 안정 추구 |
| 기질 | 정통성 중시 |
| 기질 | 신중한 수용 |
| 재능 | 학습·연구 |
| 재능 | 문서·자격 관리 |
| 재능 | 제도 해석 |
| 관계 | 스승·기관의 후원 |
| 관계 | 보호와 의존 |
| 직업 | 교육·연구·행정 |
| 직업 | 자격·문서·공공 업무 |
| 재물 | 자격·지식 기반의 안정 수입 |
| 리스크 | 준비만 길고 실행 지연 |

## 현실화

- 학력, 자격, 문서, 제도권 경력을 통해 신뢰를 얻는다.
- 복잡한 지식을 정리해 사람과 조직에 전달한다.
- 보호받는 위치에서 출발해 점차 전문 역할을 맡는다.

## 성립할 때

- 배운 것이 자격, 직책, 연구, 교육 성과로 이어진다.
- 제도와 문서를 다루는 능력이 사회적 신뢰가 된다.

## 손상될 때

- 자격과 준비는 많지만 현장에서 쓰지 못한다.
- 돈 문제로 학업, 문서, 보호 관계가 흔들린다.

# 04. 편인격 (偏印格)

**격의 중심:** 비정형 지식 · 분석 · 직관 · 특수 기술 · 독자성  
**한 줄 핵심:** 편인격은 남들이 지나치는 정보와 원리를 파고들어 특수한 지식과 전문 방식으로 쓰는 격이다.  
**성립 핵심:** 편인이 월령의 힘을 얻고 일간을 살리되 현실과 단절되지 않으며, 관살의 문제를 받아 해석하거나 재성의 조절로 결과를 만들 때 성립한다.  
**대표 성격 조합:** 살인상생 · 재제편인 · 편인용관  
**대표 손상:** 효신탈식 · 편인 과다와 고립 · 과잉 해석  
**상신 후보:** 관살의 생조 · 편인이 지나칠 때 재성의 조절  
**기신 후보:** 필요한 식신을 누르는 편인 · 현실과 단절된 인성 과다 · 검증 없는 추론  
**구신 방향:** 편인이 식신을 누르면 재성으로 편인을 조절하고, 고립된 지식에는 시장·문서·실무의 통로를 붙인다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 독자적 사고 |
| 기질 | 깊은 몰입 |
| 기질 | 낯선 것에 대한 관심 |
| 재능 | 비정형 분석 |
| 재능 | 특수 기술 습득 |
| 재능 | 숨은 원인 파악 |
| 관계 | 선택적 신뢰 |
| 관계 | 이해받기 어려운 거리감 |
| 직업 | 연구·상담·심리 |
| 직업 | 기술·기획·전문 분석 |
| 재물 | 특수 지식·전문성 수익 |
| 리스크 | 고립과 현실성 부족 |

## 현실화

- 정해진 답이 없는 문제에서 원인과 해결법을 찾는다.
- 비주류 지식, 연구, 심리, 기술을 자기 전문 분야로 만든다.
- 직관을 자료와 검증으로 연결할 때 경쟁력이 생긴다.

## 성립할 때

- 특수한 지식이 연구, 기술, 상담, 기획의 결과로 이어진다.
- 위기와 복잡한 문제를 해석해 다른 사람이 쓰게 만든다.

## 손상될 때

- 계속 분석하지만 생산과 판매로 이어지지 않는다.
- 의심과 해석이 지나쳐 사람과 현실에서 멀어진다.

# 05. 식신격 (食神格)

**격의 중심:** 생산 · 기술 · 결과물 · 생활 능력 · 지속성  
**한 줄 핵심:** 식신격은 자기 능력과 시간을 꾸준한 기술, 제품, 교육, 서비스로 만들어내는 격이다.  
**성립 핵심:** 식신이 월령의 힘을 얻고 일간에 생산할 기반이 있으며, 재성으로 판매되거나 편관을 제어하는 분명한 쓰임이 있을 때 성립한다.  
**대표 성격 조합:** 식신생재 · 식신제살  
**대표 손상:** 효신탈식 · 식다신약 · 생산과 시장의 단절  
**상신 후보:** 재성의 판매·회수 · 편관이 있을 때 식신의 제살  
**기신 후보:** 식신을 누르는 편인 · 감당하기 어려운 과다한 설기 · 판매와 회수의 부재  
**구신 방향:** 편인이 식신을 누르면 재성으로 편인을 제어하고, 일간이 약하면 생산량보다 기반을 먼저 보완한다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 꾸준함 |
| 기질 | 자기 페이스 |
| 기질 | 생활 감각 |
| 재능 | 기술 숙련 |
| 재능 | 제품·콘텐츠 생산 |
| 재능 | 교육·서비스 운영 |
| 관계 | 편안한 돌봄 |
| 관계 | 함께 일하며 쌓는 신뢰 |
| 직업 | 기술·교육·요식 |
| 직업 | 콘텐츠·제품 생산 |
| 재물 | 결과물·반복 매출 |
| 리스크 | 생산은 많고 판매가 늦음 |

## 현실화

- 자기 기술과 경험을 반복 가능한 결과물로 만든다.
- 먹고사는 능력, 생산, 교육, 서비스가 수입과 연결된다.
- 편관을 만나면 위험하고 어려운 일을 기술로 해결한다.

## 성립할 때

- 만든 것에 가격과 판매 경로가 붙는다.
- 생산량과 체력이 맞고 품질을 꾸준히 유지한다.

## 손상될 때

- 만들기만 하고 시장에 내놓지 못한다.
- 과도한 생산과 돌봄으로 자기 기반이 약해진다.

# 06. 상관격 (傷官格)

**격의 중심:** 표현 · 개선 · 비판 · 기술 혁신 · 규칙 재설계  
**한 줄 핵심:** 상관격은 기존 기준의 문제를 드러내고 더 나은 방식, 상품, 표현으로 바꾸는 격이다.  
**성립 핵심:** 상관이 월령의 힘을 얻고 재성으로 시장과 연결되거나 인성으로 정리되며, 관성을 함부로 손상하지 않을 통제 기준이 있을 때 성립한다.  
**대표 성격 조합:** 상관생재 · 상관패인 · 상관가살  
**대표 손상:** 상관견관 · 표현 과다 · 규칙과 전면 충돌  
**상신 후보:** 재성의 시장화 · 인성의 정리와 검증 · 편관이라는 해결 과제  
**기신 후보:** 보호받지 못한 정관 · 재성·인성 없는 상관 과다 · 책임 없는 표현  
**구신 방향:** 인성으로 말과 기술을 정리하거나 재성으로 결과를 회수하고, 손상된 관성의 역할을 다시 분명히 한다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 비판 의식 |
| 기질 | 독립적 표현 |
| 기질 | 간섭 거부 |
| 재능 | 문제 개선 |
| 재능 | 언변·기획 |
| 재능 | 기술 혁신 |
| 관계 | 말과 기준의 충돌 |
| 관계 | 통제받지 않으려는 태도 |
| 직업 | 콘텐츠·마케팅·기획 |
| 직업 | 기술 개선·프리랜스 |
| 재물 | 재능·상품화 수익 |
| 리스크 | 구설과 공식 관계 손상 |

## 현실화

- 불편한 규칙과 낡은 제품을 찾아 개선한다.
- 말, 기술, 기획, 디자인을 상품과 성과로 바꾼다.
- 조직 안에서는 개혁자, 밖에서는 전문 프리랜서로 쓰이기 쉽다.

## 성립할 때

- 비판 뒤에 대안, 실적, 판매 결과가 붙는다.
- 강한 표현을 자격, 자료, 계약으로 정리한다.

## 손상될 때

- 맞는 말을 하고도 상사, 기관, 고객과 관계가 끊어진다.
- 표현은 강하지만 책임과 회수 방식이 부족하다.

# 07. 정재격 (正財格)

**격의 중심:** 고정 수입 · 생활 운영 · 계약 · 책임 있는 소유

**성립 핵심:** 정재가 월령의 힘을 얻고 일간이 재물을 감당하며, 식상이 수입을 만들고 관성이 계약과 책임을 지켜줄 때 성립한다.  
**대표 성격 조합:** 식상생재 · 재생관

**상신 후보:** 식상의 생산 · 관성의 계약과 보호

**구신 방향:** 비겁의 재물 다툼에는 관성으로 기준을 세우고, 약한 일간에는 수입 확대보다 관리 능력을 먼저 보완한다.

- 책임과 정산 기준이 명확한 관계를 선호한다.

## 성립할 때

- 꾸준한 생산이 계약과 현금 회수로 이어진다.
- 수입보다 지출·세금·저축 관리가 안정된다.

## 손상될 때

- 가족, 동업자, 동료와 돈의 몫을 두고 다툰다.
- 벌이는 늘지만 책임과 고정비가 더 빠르게 커진다.

# 08. 편재격 (偏財格)

# 10. 양인격 (羊刃格)

**격의 중심:** 집중된 자기 힘 · 승부 · 결단 · 위험 감수 · 강한 통제 필요  
**한 줄 핵심:** 양인격은 극도로 강해진 자기 힘을 관살과 규율로 통제해 권한과 실전 능력으로 바꾸는 격이다.  
**성립 핵심:** 양간이 월령의 제왕지에서 양인을 얻고, 관살이 그 힘을 제어하거나 식상이 필요한 방향으로 배출할 때 성립한다.  
**대표 성격 조합:** 관살제인(制刃) · 양인가살 · 식상설인(泄刃)  
**대표 손상:** 양인무제 · 인비과다 · 충인과 권력 충돌  
**상신 후보:** 관살의 통제 · 식상의 배출 · 재관으로 이어지는 책임 구조  
**기신 후보:** 양인을 더 키우는 인비 · 힘없는 관살 · 필요한 통제를 깨는 식상  
**구신 방향:** 강한 힘에는 먼저 명령 체계와 책임 범위를 세우고, 관살이 손상되면 식상으로 함부로 치지 말고 통제 수단을 복구한다.

# 외격 보조 기준

외격은 정격보다 성립 조건이 엄격하다.  
일간이 조금 약하거나 특정 오행이 많다는 이유만으로 종격·화격·전왕격을 적용하지 않는다.

## 종격

| 구분 | 중심 상태 | 핵심 확인 |
|---|---|---|
| 종재격 | 재성이 압도하고 일간이 재물을 따름 | 일간의 유효한 근·인비가 거의 없고 재성의 방향을 거스르기 어려움 |
| 종살격 | 관살이 압도하고 일간이 권력·환경을 따름 | 일간의 유효한 근·인성이 거의 없고 관살의 힘이 맑게 모임 |
| 종아격 | 식상이 압도하고 일간이 생산·표현을 따름 | 인성·비겁의 역행이 약하고 식상이 재성 등으로 이어짐 |
| 종왕격 | 비겁과 같은 오행이 압도해 강한 기세를 따름 | 다른 오행의 독립된 제어가 약하고 같은 기운이 계절과 지지를 장악함 |
| 종강격 | 인성과 비겁이 압도해 생조의 방향을 따름 | 재관식의 독립된 역행이 약하고 인비의 생조가 끊기지 않음 |

### 종격 주의

- 약하다고 모두 종하지 않는다.
- 일간의 근 하나가 실제로 살아 있으면 가종인지 정격인지 다시 본다.
- 대운에서 역행 오행이 강하게 들어올 때 종격의 유지 여부를 다시 판단한다.

- 원래 일간의 근과 화를 방해하는 오행이 강하지 않아야 한다.
- 단순한 천간합, 합거, 합반을 화격으로 바꾸지 않는다.

### 전왕격 주의

- 한 오행이 많다는 이유만으로 성립하지 않는다.
- 월령, 방합·삼합, 투출·통근이 같은 방향으로 모여야 한다.
- 반대 오행이 독립된 근과 세력을 가지면 정격과 태과불급을 먼저 본다.
- 전왕격은 종왕격의 구체적 형태로 겹쳐 해석될 수 있으므로 중복 계산하지 않는다.

# 격국 단식 판단 금지

- 정관격이 곧 공무원은 아니다.
- 편관격이 곧 군경·사고는 아니다.
- 정인격이 곧 학자·안정된 보호자는 아니다.
- 편인격이 곧 종교인·정신 문제는 아니다.
- 식신격이 곧 요식업·복은 아니다.
- 상관격이 곧 반항·이혼은 아니다.
- 정재격이 곧 부자·안정된 결혼은 아니다.
- 편재격이 곧 사업가·큰돈은 아니다.
- 건록격이 곧 자수성가·독립 성공은 아니다.
- 양인격이 곧 사고·폭력·수술은 아니다.

격국은 역할의 이름이다. 실제 성취와 사건은 격의 힘, 상신의 성립, 기신의 손상, 구신의 복구, 대운·세운의 변화를 함께 본 뒤 판단한다.
