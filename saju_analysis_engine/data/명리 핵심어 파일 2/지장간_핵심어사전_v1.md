# 지장간 핵심어 사전 v1

지장간은 지지 안에 숨어 있는 천간이다.  
천간이 밖으로 드러난 의식과 표현이라면, 지장간은 **몸에 밴 욕망, 잠재 재능, 숨은 감정, 현실 사건의 씨앗**에 가깝다.

## 사용 원칙

- 지장간은 단순히 암기표로 쓰지 않는다.
- 본기는 지지의 중심 성질, 중기는 내부 전환점, 여기는 이전 계절의 잔기나 숨은 보조 성질로 본다.
- 지장간이 천간으로 투출하면 숨어 있던 욕망과 역할이 사회적으로 드러난다.
- 월지의 지장간은 격국·월령 판단에서 중요하고, 일지의 지장간은 배우자궁·사적 생활에서 중요하다.
- 지장간 표는 유파별 세부 차이가 있으므로, 이 문서는 실무에서 많이 쓰는 본기·중기·여기 기준으로 정리한다.

# 子 자수 지장간

**오행:** 수  
**계절:** 한겨울  
**한 줄 핵심:** 자수의 지장간은 깊은 감정과 기억이 안쪽으로 고이는 구조다.

**중심 물상:** 밤의 물속에 큰물의 흔적과 작은 물의 본심이 함께 있는 모습.

## 지장간 구성

| 역할 | 천간 | 의미 |
|---|---|---|
| 여기 | 壬(임수) | 큰물의 잔기, 넓은 정보와 이동성 |
| 본기 | 癸(계수) | 순수한 겨울 수기, 기억·감정·비밀·저장 |

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 내면성 |
| 기질 | 깊은 기억 |
| 기질 | 비밀성 |
| 재능 | 정보 감각 |
| 재능 | 해석력 |
| 재능 | 기록성 |
| 관계 | 정서적 의존 |
| 관계 | 숨은 마음 |
| 직업 | 연구·상담 |
| 직업 | 데이터·기록 |
| 재물 | 현금 흐름 |
| 리스크 | 불안과 은폐 |

## 기질

- 속으로 많이 느끼고 오래 기억한다
- 겉보다 안쪽에서 판단이 늦게 익는다
- 감정과 정보가 한곳에 고인다

## 욕망

- 안전하게 이해받고 싶은 마음
- 숨은 정보를 쥐고 싶은 욕구
- 말하지 않은 마음을 알아주길 바라는 욕망

## 재능

- 기록
- 연구
- 데이터
- 상담
- 심리 파악
- 야간 집중

## 관계

- 가까워도 비밀이 남기 쉽다
- 정서적 확인 욕구가 있다
- 상대의 작은 태도를 오래 기억한다

## 직업

- 연구
- 상담
- 기록관리
- 데이터
- 금융 보조
- 후방 업무

## 재물

- 현금 흐름과 숨은 수입에 민감하다
- 돈을 드러내기보다 안쪽에서 관리하려 한다

## 리스크

- 불안
- 은폐
- 우유부단
- 관계 비밀
- 수면과 리듬 문제

## 월지에서의 쓰임

- 월지 子의 지장간은 월령의 순수한 수기로 작동한다. 조후상 火의 필요가 커진다.

## 일지에서의 쓰임

- 일지 子는 배우자궁과 사적 생활에 감정·기억·비밀성을 만든다.

## 운에서 들어올 때

- 운에서 子가 오면 숨은 감정, 돈의 흐름, 연애 감정, 내부 변화가 올라온다.

## 투출될 때

- 壬이 투출하면 큰 정보·이동성이 드러나고, 癸가 투출하면 감수성·기억·비밀이 밖으로 나온다.

## 좋아지는 조건

- 감정과 정보를 기록·상담·연구로 바꿀 때
- 숨은 돈의 흐름을 명확히 관리할 때

## 나빠지는 조건

- 비밀이 많아지고, 불안 때문에 행동을 미룰 때

# 丑 축토 지장간

**오행:** 토  
**계절:** 겨울 끝  
**한 줄 핵심:** 축토의 지장간은 감정과 가치와 생활 기반이 얼어붙은 창고다.

**중심 물상:** 차가운 창고 안에 물과 금속과 흙이 함께 보관된 모습.

## 지장간 구성

| 역할 | 천간 | 의미 |
|---|---|---|
| 여기 | 癸(계수) | 겨울 수기의 잔류, 불안·기억·저장 |
| 중기 | 辛(신금) | 차가운 금기, 품질·분별·응축 |
| 본기 | 己(기토) | 얼어붙은 습토, 보관·생활·실무 |

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 신중함 |
| 기질 | 저장성 |
| 기질 | 인내심 |
| 재능 | 축적 능력 |
| 재능 | 정밀 실무 |
| 재능 | 재고 관리 |
| 관계 | 늦은 신뢰 |
| 관계 | 책임 부담 |
| 직업 | 회계·관리 |
| 직업 | 창고·자산 |
| 재물 | 저축·보존 |
| 리스크 | 정체와 미련 |

## 기질

- 쉽게 열지 않고 오래 저장한다
- 느리지만 버티는 힘이 있다
- 감정과 돈을 동시에 보관하려 한다

## 욕망

- 안전한 기반을 만들고 싶은 욕망
- 손실을 피하려는 마음
- 오래 남을 가치를 지키고 싶은 욕구

## 재능

- 저축
- 회계
- 재고 관리
- 품질 점검
- 장기 보관
- 위험 회피

## 관계

- 믿기까지 시간이 걸린다
- 정이 들면 쉽게 끊지 않는다
- 표현은 느려도 책임은 오래 간다

## 직업

- 회계
- 자산관리
- 물류·창고
- 행정
- 품질관리
- 농축산·식품

## 재물

- 저축과 보존에 강하다
- 묶인 돈과 장기 자산에 인연이 있다
- 현금화는 느릴 수 있다

## 리스크

- 미련
- 정체
- 불안 저장
- 속병
- 오래된 문제의 누적

## 월지에서의 쓰임

- 월지 丑은 겨울의 끝과 축적의 계절이다. 火와 木이 들어와야 얼어붙은 것이 움직인다.

## 일지에서의 쓰임

- 일지 丑은 생활과 배우자궁에 안정 욕구, 말없는 책임, 늦은 개방을 만든다.

## 운에서 들어올 때

- 운에서 丑이 오면 자산 정리, 묶인 돈, 오래된 가족·부동산 문제가 올라온다.

## 투출될 때

- 癸가 투출하면 감정·기억이, 辛이 투출하면 기준·품질이, 己가 투출하면 생활 책임이 드러난다.

## 좋아지는 조건

- 느린 축적이 자산이 될 때
- 정밀 실무와 보존 능력이 인정받을 때

## 나빠지는 조건

- 오래된 문제를 닫아두기만 하고 변화시키지 못할 때

# 寅 인목 지장간

**오행:** 목  
**계절:** 초봄  
**한 줄 핵심:** 인목의 지장간은 기반을 딛고, 드러내고, 주체적으로 밀고 나가는 구조다.

**중심 물상:** 새벽 숲에서 땅을 딛고 햇빛을 받아 큰 나무가 올라오는 모습.

## 지장간 구성

| 역할 | 천간 | 의미 |
|---|---|---|
| 여기 | 戊(무토) | 발동을 받치는 현실 기반 |
| 중기 | 丙(병화) | 봄기운을 밖으로 드러내는 표현과 상승 |
| 본기 | 甲(갑목) | 초봄의 주체적 성장과 개척 |

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 개척성 |
| 기질 | 자존심 |
| 기질 | 추진력 |
| 재능 | 시작 능력 |
| 재능 | 리더십 |
| 재능 | 표현 기획 |
| 관계 | 주도성 |
| 관계 | 경쟁 의식 |
| 직업 | 창업·기획 |
| 직업 | 교육·리더 |
| 재물 | 초기 투자 |
| 리스크 | 성급한 확장 |

## 기질

- 시작하려는 힘이 강하다
- 기반과 표현과 주체성이 함께 있다
- 정체된 상황을 답답해한다

## 욕망

- 자기 길을 열고 싶은 욕망
- 앞에서 이끌고 싶은 마음
- 성장과 사회적 상승을 동시에 원하는 욕구

## 재능

- 창업
- 기획
- 교육
- 리더십
- 초기 개척
- 사람을 움직이는 힘

## 관계

- 관계에서 주도권이 강하다
- 상대에게 성장과 변화를 요구한다
- 경쟁과 협력이 같이 생긴다

## 직업

- 창업
- 기획
- 교육
- 영업 개척
- 조직 리더
- 브랜드 시작

## 재물

- 초기 투자와 성장형 수입
- 기회가 먼저 보이고 회수는 늦을 수 있다

## 리스크

- 성급함
- 마무리 부족
- 권위 충돌
- 무리한 시작
- 과한 자신감

## 월지에서의 쓰임

- 월지 寅은 봄의 발동이다. 戊·丙·甲이 모두 살아나면 현실 기반, 표현, 주체성이 함께 열린다.

## 일지에서의 쓰임

- 일지 寅은 배우자궁과 사적 생활에 독립성과 주도권을 넣는다.

## 운에서 들어올 때

- 운에서 寅이 오면 새 출발, 이직, 사업 발동, 이동과 경쟁이 올라온다.

## 투출될 때

- 戊가 투출하면 기반과 책임, 丙이 투출하면 표현과 명예, 甲이 투출하면 자기 방향이 드러난다.

## 좋아지는 조건

- 기반·표현·주체성이 함께 이어질 때
- 시작한 일을 사회적 성과로 만들 때

## 나빠지는 조건

- 시작만 많고 책임과 마무리가 없을 때

# 卯 묘목 지장간

**오행:** 목  
**계절:** 중춘  
**한 줄 핵심:** 묘목의 지장간은 큰 방향의 흔적을 품은 순수한 관계성과 미감이다.

**중심 물상:** 봄꽃과 풀밭 속에 큰 줄기의 흔적이 남아 있는 모습.

## 지장간 구성

| 역할 | 천간 | 의미 |
|---|---|---|
| 여기 | 甲(갑목) | 이전 계절에서 이어진 목의 큰 방향 |
| 본기 | 乙(을목) | 순수한 음목, 관계·미감·부드러운 확장 |

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 호감성 |
| 기질 | 미감 |
| 기질 | 눈치 |
| 재능 | 관계 조율 |
| 재능 | 디자인 감각 |
| 재능 | 상담성 |
| 관계 | 도화성 |
| 관계 | 선택 욕구 |
| 직업 | 디자인·상담 |
| 직업 | 서비스·교육 |
| 재물 | 고객 반응 수익 |
| 리스크 | 비교와 질투 |

## 기질

- 사람 반응과 분위기를 잘 본다
- 부드럽지만 속의 취향이 분명하다
- 호감과 선별성이 같이 있다

## 욕망

- 선택받고 싶은 욕망
- 관계 속에서 자기 자리를 얻고 싶은 마음
- 아름답고 세련된 모습으로 인정받고 싶은 욕구

## 재능

- 디자인
- 미용
- 상담
- 서비스
- 고객 반응 파악
- 브랜딩 감각

## 관계

- 호감과 인기, 비교와 질투가 같이 움직인다
- 좋아해도 쉽게 마음을 다 열지는 않는다

## 직업

- 디자인
- 미용
- 패션
- 상담
- 교육
- 고객관리
- 브랜딩

## 재물

- 취향과 이미지, 고객 반응으로 돈이 된다
- 관계 비용과 체면 비용이 커질 수 있다

## 리스크

- 질투
- 비교
- 구설
- 관계 피로
- 우회적 불만

## 월지에서의 쓰임

- 월지 卯는 목의 순수한 번성이다. 酉金의 평가, 丁火의 표현, 己土의 생활화가 관건이다.

## 일지에서의 쓰임

- 일지 卯는 배우자궁에 호감, 도화, 취향, 예민한 관계성을 넣는다.

## 운에서 들어올 때

- 운에서 卯가 오면 사람, 고객, 이성, 이미지, 취향, 관계 확장이 올라온다.

## 투출될 때

- 甲이 투출하면 큰 방향과 자존심이, 乙이 투출하면 미감·관계·호감성이 밖으로 나온다.

## 좋아지는 조건

- 관계성과 미감이 전문성으로 바뀔 때
- 호감이 고객과 브랜드로 연결될 때

## 나빠지는 조건

- 비교와 질투가 관계를 갉아먹을 때

**중심 물상:** 습한 땅 속에 풀뿌리와 물과 큰 흙이 함께 들어 있는 모습.

## 지장간 구성

| 역할 | 천간 | 의미 |
|---|---|---|

# 酉 유금 지장간

**오행:** 금  
**계절:** 중추  
**한 줄 핵심:** 유금의 지장간은 큰 절단의 흔적을 품은 순수한 평가와 완성도다.

**중심 물상:** 잘 세공된 금속과 보석 속에 큰 칼의 잔상이 남아 있는 모습.

## 지장간 구성

| 역할 | 천간 | 의미 |
|---|---|---|
| 여기 | 庚(경금) | 큰 금의 잔류, 강한 절단과 판단 |
| 본기 | 辛(신금) | 순수한 음금, 미감·품질·완성도·평가 |

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 미감 발달 |
| 기질 | 자존심 |
| 기질 | 평가 의식 |
| 재능 | 품질 감각 |
| 재능 | 브랜딩 |
| 재능 | 정밀함 |
| 관계 | 선별성 |
| 관계 | 도화성 |
| 직업 | 디자인·심사 |
| 직업 | 금융·품질 |
| 재물 | 고급 가치 |
| 리스크 | 구설과 비교 |

## 기질

- 작은 차이와 완성도를 잘 본다
- 외모·이미지·말투에 민감하다
- 값어치와 수준을 따진다

## 욕망

- 좋은 평가를 받고 싶은 욕망
- 가치 있는 사람으로 보이고 싶은 마음
- 세련되고 완성도 있는 결과를 원하는 욕구

## 재능

- 디자인
- 미용
- 편집
- 품질관리
- 브랜딩
- 심사

## 관계

- 매력과 선별성이 함께 있다
- 비교와 질투가 관계에 들어오기 쉽다
- 예의와 수준을 중요하게 본다

## 직업

- 디자인
- 미용
- 패션
- 금융
- 편집
- 브랜딩
- 품질관리

## 재물

- 고급 취향과 희소성이 돈이 된다
- 품질과 이미지로 가격을 받는다

## 리스크

- 비교심
- 질투
- 구설
- 까다로움
- 이미지 비용
- 관계 피로

## 월지에서의 쓰임

- 월지 酉는 금의 완성도와 평가가 강한 자리다. 화로 단련되고 수로 씻기면 지성과 매력이 살아난다.

## 일지에서의 쓰임

- 일지 酉는 배우자궁에 선별성, 도화성, 예민한 평가를 넣는다.

## 운에서 들어올 때

- 운에서 酉가 오면 이미지, 평가, 이성, 고객 반응, 품질, 금융·계약 문제가 올라온다.

## 투출될 때

- 庚이 투출하면 큰 결단과 절단, 辛이 투출하면 세밀한 미감과 평가가 드러난다.

## 좋아지는 조건

- 미감과 품질이 전문성으로 연결될 때
- 브랜드와 고객 반응을 돈으로 바꿀 때

## 나빠지는 조건

- 비교와 평가가 관계를 갉아먹을 때

# 戌 술토 지장간

**오행:** 토  
**계절:** 가을 끝  
**한 줄 핵심:** 술토의 지장간은 기준과 불씨와 책임이 마른 흙 속에 저장된 구조다.

**중심 물상:** 마른 산과 창고 속에 보석과 불씨와 큰 흙이 들어 있는 모습.

## 지장간 구성

| 역할 | 천간 | 의미 |
|---|---|---|
| 여기 | 辛(신금) | 가을 금의 잔류, 기준·품질·평가 |
| 중기 | 丁(정화) | 묘고 속 불씨, 감정·신념·분노 |
| 본기 | 戊(무토) | 마른 토, 책임·저장·방어 |

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 책임감 |
| 기질 | 신념 |
| 기질 | 고집 |
| 재능 | 보호 능력 |
| 재능 | 위기 수습 |
| 재능 | 기준 정리 |
| 관계 | 의리 |
| 관계 | 경계심 |
| 직업 | 관리·법 |
| 직업 | 보안·부동산 |
| 재물 | 묵은 자산 |
| 리스크 | 분노와 고립 |

## 기질

- 책임과 신념이 강하다
- 속에 불만과 분노를 오래 품는다
- 자기 사람과 자기 영역을 지키려 한다

## 욕망

- 명예와 의리를 지키고 싶은 욕망
- 자기 영역을 침범당하지 않으려는 마음
- 오래된 것을 지키려는 욕구

## 재능

- 보호
- 보안
- 법과 규정
- 자산 보존
- 위기 수습

## 관계

- 자기 사람을 오래 지킨다
- 배신과 무례를 잊기 어렵다
- 가까워도 경계가 남는다

## 직업

- 법무
- 보안
- 부동산
- 관리
- 군경
- 시설관리

## 재물

- 묵은 자산과 부동산
- 장기 재산 관리
- 권리와 문서

## 리스크

- 분노 저장
- 고립
- 권위 충돌
- 관계 단절
- 묵은 문제

## 월지에서의 쓰임

- 월지 戌은 가을 끝의 마른 토다. 辛의 기준, 丁의 불씨, 戊의 책임 중 무엇이 열리는지가 중요하다.

## 일지에서의 쓰임

- 일지 戌은 배우자궁에 의리, 경계, 책임, 오래 묵은 불만을 넣는다.

## 운에서 들어올 때

- 운에서 戌이 오면 부동산, 법적 문제, 묵은 관계, 책임, 가족·조직의 오래된 문제가 올라온다.

## 투출될 때

- 辛이 투출하면 기준·평가, 丁이 투출하면 숨은 감정·분노, 戊가 투출하면 책임·자산이 드러난다.

## 좋아지는 조건

- 신념과 책임이 직업적 신뢰가 될 때
- 묵은 자산과 문제를 정리할 때

## 나빠지는 조건

- 분노와 고집을 오래 저장할 때

# 亥 해수 지장간

**오행:** 수  
**계절:** 초겨울  
**한 줄 핵심:** 해수의 지장간은 큰물 속에 현실의 경계와 새 시작의 씨앗이 함께 있는 구조다.

**중심 물상:** 초겨울의 바다 속에 땅의 흔적과 큰 나무의 씨앗이 잠겨 있는 모습.

## 지장간 구성

| 역할 | 천간 | 의미 |
|---|---|---|
| 여기 | 戊(무토) | 이전 계절의 토기, 경계와 현실 기반 |
| 중기 | 甲(갑목) | 물속의 씨앗, 새 시작과 성장 가능성 |
| 본기 | 壬(임수) | 큰 물, 이동·감정·철학성 |

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 깊은 감정 |
| 기질 | 이동성 |
| 기질 | 철학성 |
| 재능 | 직관 |
| 재능 | 상담 감각 |
| 재능 | 외부 확장 |
| 관계 | 정서적 깊이 |
| 관계 | 거리감 |
| 직업 | 상담·연구 |
| 직업 | 무역·이동 |
| 재물 | 유동 수익 |
| 리스크 | 도피와 비밀 |

## 기질

- 속이 깊고 감정의 양이 많다
- 끝과 시작을 함께 품는다
- 한곳에 묶이기보다 흐르고 싶어 한다

## 욕망

- 깊이 이해받고 싶은 욕망
- 새 길을 찾고 싶은 마음
- 자유롭게 흐르고 싶은 욕구

## 재능

- 상담
- 철학
- 종교·역술
- 무역
- 외국어
- 이동·확장

## 관계

- 깊게 연결되지만 거리도 필요하다
- 외로움과 자유 욕구가 함께 있다
- 관계에 비밀성이 생기기 쉽다

## 직업

- 상담
- 연구
- 철학
- 무역
- 물류
- 여행
- 외국어·해외

## 재물

- 유동 수익, 부업, 이동과 거래의 돈
- 감정과 경험의 상품화

## 리스크

- 도피
- 비밀
- 관계 복잡성
- 감정 침잠
- 현실 지연

## 월지에서의 쓰임

- 월지 亥는 초겨울의 큰물이다. 壬水가 중심이고, 甲木이 씨앗처럼 들어 있어 다음 성장의 가능성을 품는다.

## 일지에서의 쓰임

- 일지 亥는 배우자궁에 깊은 정서, 이동성, 비밀성, 자유 욕구를 넣는다.

## 운에서 들어올 때

- 운에서 亥가 오면 이동, 해외, 철학·상담, 감정, 비밀, 새로운 시작과 끝맺음이 올라온다.

## 투출될 때

- 戊가 투출하면 경계·현실, 甲이 투출하면 새 방향, 壬이 투출하면 큰 물과 이동성이 드러난다.

## 좋아지는 조건

- 깊은 감정이 상담·연구·이동성 직업으로 쓰일 때
- 끝낸 뒤 새 길을 열 때

## 나빠지는 조건

- 현실을 피하고 감정과 비밀 속으로 들어갈 때
