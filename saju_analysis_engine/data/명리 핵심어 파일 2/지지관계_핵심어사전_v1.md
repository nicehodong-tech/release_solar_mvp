# 지지 관계 핵심어 사전 v1

이 문서는 지지 관계를 긴 해설이 아니라 **핵심어 자산**으로 정리한 사전이다.  
각 항목은 `기질 / 재능 / 관계 / 직업 / 리스크` 태그가 붙은 12개 핵심어로 구성한다.

## 사용 기준

- 단식 판단 재료로 사용한다.
- 항목 하나만으로 이혼, 사고, 파산, 질병을 확정하지 않는다.
- 최종 통변에서는 월령, 궁성, 십신, 강약, 대운·세운을 함께 본다.
- 같은 지지 조합에 합·충·형·파·해 등이 겹치면 어느 하나로 나머지를 지우지 않는다.
- 원진·귀문·암합은 합·충·형·파·해보다 판단 비중을 낮춘다.
- 암합은 학파별 조합 차이가 있어, 이 문서에서는 지장간 합을 기준으로 널리 쓰는 4개 조합만 수록한다.
- 이 사전은 고객용 문장 생성 전 단계의 핵심어 뱅크다.

## 수록 범위

- 육합 6개
- 삼합 4개
- 삼합 내 반합·공합 12개
- 방합 4개
- 육충 6개
- 형 13개
- 육해 6개
- 육파 6개
- 원진 6개
- 귀문 6개
- 암합 4개

|---|---|---|
| 원진 | 애증·서운함·끊기 어려움 | 악연·원한 관계 |
| 귀문 | 과민한 감지·몰입·숨은 뜻 해석 | 정신질환·영적 현상 |
| 암합 | 말하지 않은 이해관계·사적 합의·숨은 끌림 | 불륜·불법 거래 |
