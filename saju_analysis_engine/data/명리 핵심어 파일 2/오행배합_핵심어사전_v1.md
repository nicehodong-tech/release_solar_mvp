# 오행 배합 핵심어 사전 v1

오행 배합은 단일 오행보다 실제 통변에 가깝다.  
이 사전은 2오행 10개, 3오행 10개를 핵심어 중심으로 정리한 자료다.

## 기본 원칙

- 배합은 오행을 더하는 것이 아니라, **이어짐·막힘·긴장·전환**을 보는 것이다.
- 같은 배합도 월령, 조후, 강약, 천간 노출, 지지 기반, 십신으로 쓰임이 달라진다.
- 아래 핵심어는 고객 문장 자체가 아니라 통변과 섹션 선택의 재료다.

## 수록 범위

- 2오행: 목화, 목토, 목금, 목수, 화토, 화금, 화수, 토금, 토수, 금수
- 3오행: 목화토, 목화금, 목화수, 목토금, 목토수, 목금수, 화토금, 화토수, 화금수, 토금수

# 2오행

## 목화(木火)

**한 줄 핵심:** 목화는 생각과 가능성을 밖으로 드러내는 배합이다.

**작용 구조:** 목생화. 성장한 것이 표현과 인지로 이어진다.

### 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 성장 욕구 |
| 기질 | 표현 욕구 |
| 기질 | 낙관성 |
| 재능 | 기획력 |
| 재능 | 발표력 |
| 재능 | 교육 감각 |
| 관계 | 호감 확장 |
| 관계 | 인정 욕구 |
| 직업 | 교육·콘텐츠 |
| 직업 | 홍보·기획 |
| 재물 | 아이디어 수익화 |
| 리스크 | 마무리 부족 |

### 기질

- 생각이 빠르게 밖으로 나온다
- 가능성을 말로 키운다
- 사람들의 반응에 힘을 얻는다
- 지루하고 닫힌 환경을 답답해한다

### 재능

- 기획
- 강의
- 표현
- 콘텐츠
- 새로운 프로젝트의 시작

### 관계

- 사람을 설득하고 동기부여한다
- 호감과 주목을 얻기 쉽다
- 관계가 넓어지지만 얕아질 수 있다

### 직업

- 교육
- 콘텐츠
- 홍보
- 마케팅
- 브랜드 초기 기획
- 강의

### 재물

- 아이디어와 말이 돈이 된다
- 인지도와 평판이 수입으로 이어진다
- 초기 확장 비용이 커질 수 있다

### 리스크

- 말이 앞서고 실행과 회수가 늦는다
- 시작은 많고 끝은 흐려질 수 있다
- 체면과 인정 욕구로 무리한다

### 좋아지는 조건

- 방향이 분명하고 결과물까지 이어질 때
- 사람 앞에서 말할 무대가 있을 때
- 기획과 표현에 가격이 붙을 때

### 나빠지는 조건

- 계획만 크고 책임자가 없을 때
- 칭찬과 관심만 좇을 때
- 현실 계산 없이 일을 벌일 때

## 목토(木土)

**한 줄 핵심:** 목토는 성장하려는 힘과 현실의 무게가 맞서는 배합이다.

**작용 구조:** 목극토. 계획과 확장이 생활·자산·책임을 건드린다.

### 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 진취성 |
| 기질 | 현실 부담 |
| 기질 | 고집 |
| 재능 | 개척력 |
| 재능 | 생활 설계 |
| 재능 | 조직 구축 |
| 관계 | 책임 충돌 |
| 관계 | 가족 부담 |
| 직업 | 부동산·기획 |
| 직업 | 운영 개척 |
| 재물 | 자산 확장 |
| 리스크 | 무리한 책임 |

### 기질

- 앞으로 가고 싶지만 현실 조건을 계속 의식한다
- 가족·돈·조직의 부담 속에서 성장한다
- 자기 계획을 실제 기반으로 만들려 한다

### 재능

- 토지·자산 기획
- 조직 성장
- 현실을 고치는 능력
- 장기 기반 만들기

### 관계

- 관계에서 책임을 많이 본다
- 상대가 성장하지 않으면 답답해한다
- 가족과 자기 욕망이 부딪히기 쉽다

### 직업

- 부동산
- 건설
- 교육 사업
- 조직 운영
- 행정 기획
- 지역 개발

### 재물

- 돈을 자산과 기반으로 바꾸려 한다
- 확장 욕구가 투자로 이어진다
- 초기 비용과 고정비가 커질 수 있다

### 리스크

- 현실보다 계획을 앞세운다
- 가족·사업·부동산에 돈이 묶인다
- 고집으로 손절이 늦다

### 좋아지는 조건

- 계획에 예산과 기간이 붙을 때
- 성장과 안정이 함께 설계될 때
- 자기 땅과 자기 시스템이 생길 때

### 나빠지는 조건

- 무리한 확장과 과한 대출이 겹칠 때
- 가족 책임과 사업 욕심을 동시에 떠안을 때
- 고집으로 이미 틀어진 일을 계속 끌 때

## 목금(木金)

**한 줄 핵심:** 목금은 성장하려는 힘과 자르려는 기준이 부딪히는 배합이다.

**작용 구조:** 금극목. 가능성과 분별, 성장과 절단이 맞선다.

### 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 긴장감 |
| 기질 | 경쟁심 |
| 기질 | 분별 의식 |
| 재능 | 교정 능력 |
| 재능 | 전략 감각 |
| 재능 | 품질 개선 |
| 관계 | 평가 긴장 |
| 관계 | 자존심 충돌 |
| 직업 | 법·기술 |
| 직업 | 품질·심사 |
| 재물 | 선별 투자 |
| 리스크 | 관계 절단 |

### 기질

- 자라려는 마음과 평가받는 압박이 함께 있다
- 고치고 다듬는 힘이 강하다
- 부드럽게 시작해도 기준이 들어오면 날카로워진다

### 재능

- 교정
- 심사
- 전략
- 품질 개선
- 불필요한 것을 잘라내는 능력

### 관계

- 사람을 키우려 하면서도 평가한다
- 존중받지 못하면 관계를 끊는다
- 경쟁과 비교가 관계에 들어오기 쉽다

### 직업

- 법률
- 품질 관리
- 디자인 검수
- 기술 개발
- 전략 기획
- 편집

### 재물

- 가치 있는 것을 고르고 불필요한 손실을 줄인다
- 다만 판단이 늦으면 기회를 놓친다

### 리스크

- 비판이 많아진다
- 가능성을 너무 일찍 자른다
- 자존심 싸움으로 관계와 일을 끊는다

### 좋아지는 조건

- 평가 기준이 명확할 때
- 성장한 것을 다듬어 완성해야 할 때
- 기술과 품질로 인정받을 때

### 나빠지는 조건

- 비교와 비판만 남을 때
- 상대의 가능성을 전부 부족함으로 볼 때
- 자존심 때문에 협업을 끊을 때

## 목수(木水)

**한 줄 핵심:** 목수는 축적한 감각과 지식에서 새로운 방향을 뽑아내는 배합이다.

**작용 구조:** 수생목. 기억·정보·감수성이 성장과 계획을 밀어준다.

### 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 상상력 |
| 기질 | 학습 욕구 |
| 기질 | 유연함 |
| 재능 | 기획력 |
| 재능 | 언어 감각 |
| 재능 | 상담 감각 |
| 관계 | 이해심 |
| 관계 | 정서 연결 |
| 직업 | 교육·연구 |
| 직업 | 글·상담 |
| 재물 | 지식 수익 |
| 리스크 | 현실 지연 |

### 기질

- 생각이 깊고 가능성을 오래 본다
- 감정과 정보가 계획으로 자란다
- 사람의 이야기를 듣고 방향을 제시하려 한다

### 재능

- 언어
- 상담
- 기획
- 교육
- 연구
- 창작의 씨앗

### 관계

- 상대의 사정을 이해한다
- 사람의 가능성을 오래 본다
- 관계가 정서와 미래 계획으로 연결된다

### 직업

- 교육
- 상담
- 연구
- 출판
- 기획
- 심리·철학
- 콘텐츠

### 재물

- 지식과 경험이 돈이 된다
- 정보·교육·상담 수입에 강하다
- 수익화는 늦어질 수 있다

### 리스크

- 생각만 커지고 실행이 늦다
- 감정과 이상에 오래 머문다
- 현실 계산이 부족하면 돈이 남지 않는다

### 좋아지는 조건

- 배움이 결과물로 이어질 때
- 상담·교육·글로 표현될 때
- 정서와 계획이 함께 살아날 때

### 나빠지는 조건

- 가능성만 보고 사람과 일을 오래 붙잡을 때
- 계획은 많은데 마감과 가격이 없을 때
- 현실보다 해석이 앞설 때

## 화토(火土)

**한 줄 핵심:** 화토는 드러난 것을 책임과 결과로 굳히는 배합이다.

**작용 구조:** 화생토. 표현·평판·열정이 생활·조직·자산으로 내려앉는다.

### 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 책임 의식 |
| 기질 | 체면 |
| 기질 | 현실화 욕구 |
| 재능 | 운영 능력 |
| 재능 | 관리력 |
| 재능 | 공적 설득 |
| 관계 | 보호 본능 |
| 관계 | 가족 중심 |
| 직업 | 관리·교육 |
| 직업 | 공공·운영 |
| 재물 | 기반 형성 |
| 리스크 | 과로와 고집 |

### 기질

- 말과 의욕을 실제 책임으로 바꾼다
- 사람들 앞에서 인정받고 생활 기반을 만들려 한다
- 보호자와 책임자 역할을 자주 맡는다

### 재능

- 운영
- 교육
- 관리
- 공적 설득
- 조직 안정화

### 관계

- 사람을 챙기고 보호한다
- 가족·조직을 자신의 책임으로 느낀다
- 사랑이 돌봄과 통제로 바뀔 수 있다

### 직업

- 교육
- 공공기관
- 관리직
- 운영
- 부동산 관리
- 조직 책임자

### 재물

- 평판과 직위가 돈으로 이어진다
- 자산과 생활 기반을 만든다
- 가족·체면·조직 지출이 크다

### 리스크

- 책임을 너무 많이 짊어진다
- 체면을 위해 무리한다
- 말보다 부담이 커지고 몸이 무거워진다

### 좋아지는 조건

- 권한과 책임이 함께 있을 때
- 평판이 자산과 직위로 연결될 때
- 조직에서 중심 역할을 받을 때

### 나빠지는 조건

- 책임만 있고 결정권이 없을 때
- 체면 때문에 지출과 과로가 커질 때
- 가족과 조직을 전부 떠안을 때

## 화금(火金)

**한 줄 핵심:** 화금은 드러내는 힘과 다듬는 기준이 맞부딪히는 배합이다.

**작용 구조:** 화극금. 표현·열·평판이 품질·기준·자존심을 건드린다.

### 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 평가 민감 |
| 기질 | 자존심 |
| 기질 | 세련 욕구 |
| 재능 | 브랜딩 |
| 재능 | 무대 감각 |
| 재능 | 완성도 |
| 관계 | 시선 긴장 |
| 관계 | 비교 의식 |
| 직업 | 이미지 산업 |
| 직업 | 심사·디자인 |
| 재물 | 평판 수익 |
| 리스크 | 구설과 체면 손상 |

### 기질

- 보이고 싶으면서도 허술하게 보이기는 싫다
- 말과 이미지, 평가와 자존심이 함께 움직인다
- 완성도와 평판에 민감하다

### 재능

- 브랜딩
- 디자인
- 대중 노출
- 심사
- 무대에서의 완성도

### 관계

- 사람들의 시선에 민감하다
- 관계에서 비교와 자존심이 생긴다
- 칭찬과 비판 모두 강하게 받아들인다

### 직업

- 패션
- 미용
- 브랜딩
- 방송
- 디자인
- 심사
- 고급 서비스

### 재물

- 이미지와 평판이 돈이 된다
- 브랜드와 품질로 값을 받는다
- 외형과 체면 비용이 커진다

### 리스크

- 말실수와 구설
- 비교심
- 과시
- 평가 불안
- 관계에서 날이 선다

### 좋아지는 조건

- 보이는 결과물의 완성도를 높일 때
- 브랜드·이미지·품질이 돈으로 연결될 때
- 무대와 기준이 함께 있을 때

### 나빠지는 조건

- 체면과 비교심이 앞설 때
- 평판 때문에 무리한 소비를 할 때
- 말과 이미지가 실제 실력보다 앞설 때

## 화수(火水)

**한 줄 핵심:** 화수는 드러내려는 마음과 숨기려는 마음이 동시에 움직이는 배합이다.

**작용 구조:** 수극화. 표현과 감정, 노출과 비밀, 인기와 불안이 충돌한다.

### 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 기질 | 감정 기복 |
| 기질 | 내면 불안 |
| 기질 | 이중성 |
| 재능 | 분위기 감지 |
| 재능 | 심리 감각 |
| 재능 | 극적 표현 |
| 관계 | 밀고 당김 |
| 관계 | 애정 확인 |
| 직업 | 상담·콘텐츠 |
| 직업 | 대외·심리 |
| 재물 | 인기 수익 |
| 리스크 | 평판 흔들림 |

### 기질

- 겉으로는 밝아도 안에서는 불안이 움직인다
- 사람들 앞에 서고 싶지만 속마음은 숨긴다
- 감정과 이미지의 온도차가 크다

### 재능

- 감정 표현
- 심리 감지
- 극적 스토리텔링
- 상담
- 대중 반응 파악

### 관계
