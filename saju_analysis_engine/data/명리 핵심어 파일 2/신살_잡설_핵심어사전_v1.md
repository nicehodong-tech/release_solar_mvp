# 신살·잡설 핵심어 사전 v1
고객 상품용으로 쓸 수 있는 **핵심 신살·12신살**과, 내부 보관·검토용으로만 둘 **잡설**을 분리했다.
각 항목은 해설문이 아니라 `기질 / 재능 / 관계 / 직업 / 리스크` 태그를 붙인 12개 핵심어 사전이다.
요청에 따라 `단독 사용 가능 여부`, `단독 사용 금지 판단` 항목은 넣지 않았다.

## 사용 구분

| 사용 구분 | 의미 |
|---|---|
| 상품용 | 고객 결과지, 블로그, 섹션 문장에 사용할 수 있는 핵심 신살 |
| 내부보관용 | 엔진·자료 보관·연구용. 고객 문장 기본 미사용 |
