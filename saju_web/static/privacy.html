AI 사주 : 이현

개인정보 처리방침

본 서비스는 사주 분석 결과를 생성하기 위해 사용자가 입력한 생년월일, 태어난 시간, 성별 정보를 사용합니다. 입력 정보는 분석 결과를 생성하는 데 사용되며, 별도의 회원 가입이나 결제 정보 수집 없이 운영됩니다.

수집하는 정보

생년월일, 태어난 시간, 성별, 서비스 이용 과정에서 서버가 자동으로 처리하는 접속 정보가 포함될 수 있습니다.

이용 목적

입력된 정보는 만세력 기반 사주 분석 결과를 생성하고, 서비스 오류를 확인하며, 안정적인 운영을 유지하기 위해 사용됩니다.

보관

현재 MVP는 회원 계정과 별도 데이터베이스를 운영하지 않습니다. 사용자가 입력한 정보는 분석 응답 생성 과정에서만 사용됩니다.

외부 서비스

향후 쿠팡 파트너스, 통계, 결제 기능이 추가될 경우 해당 외부 서비스의 정책에 따라 필요한 정보가 처리될 수 있으며, 변경 사항은 이 페이지에 반영합니다.

문의
