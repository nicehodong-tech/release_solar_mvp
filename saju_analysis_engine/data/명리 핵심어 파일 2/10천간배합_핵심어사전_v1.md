# 10천간 배합 핵심어 사전 v1

10천간 배합은 천간 두 글자가 겉으로 드러내는 의식, 욕망, 재능, 사회적 작용을 보는 사전이다.  
이 문서는 10천간의 2글자 조합 45개를 정리했다.

## 사용 원칙

- 천간 배합은 오행 배합보다 구체적이다. 같은 목화라도 甲丙, 甲丁, 乙丙, 乙丁은 다르게 본다.
- 두 천간의 작용은 월령, 지지 기반, 투출·통근, 십신 관계에 따라 달라진다.
- 천간합 5개는 해당 조합 안에 함께 표시했다.
- 이 사전은 고객 문장 자체가 아니라 통변과 섹션 선택을 위한 핵심어 자산이다.

## 천간합 5개

| 합 | 합화 |
|---|---|
| 甲己 | 土 |
| 乙庚 | 金 |
| 丙辛 | 水 |
| 丁壬 | 木 |
| 戊癸 | 火 |
