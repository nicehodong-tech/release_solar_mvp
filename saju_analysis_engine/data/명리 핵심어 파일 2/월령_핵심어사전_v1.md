# 월령 핵심어 사전 v1

월령은 태어난 달의 지지를 성격표처럼 읽는 항목이 아니다.  
사람이 놓인 계절 환경, 그 환경이 공통으로 요구하는 업무, 실제로 발휘해야 할 능력, 사회적 쓰임의 출발점을 보는 기준이다.

## 사용 원칙

- 월령은 단순히 가장 강한 글자가 아니라 사람이 놓인 환경이다.
- 당령은 그 환경이 공통으로 요구하는 업무와 임무다.
- 사령은 절입 후 경과와 월률분야에 따라 개인에게 구체적으로 주어진 기능이다.
- 아래 지장간 분화는 사령 후보의 역할을 정리한 것이며, 출생일 없이 실제 사령을 확정하지 않는다.
- 조화는 환경에 맞는 능력을 갖추는 과정이고, 조후는 그 능력이 제대로 쓰일 온도와 습도의 조건이다.
- 조후상 필요한 오행을 격국용신·억부용신과 같은 뜻으로 사용하지 않는다.
- 실제 판단에서는 일간의 강약, 투출·통근, 지지 배합, 격국, 병약, 대운·세운을 함께 본다.
- 이 사전은 고객 문장 자체가 아니라 통변과 섹션 선택을 위한 핵심어 자산이다.

## 수록 형식

- 12월령: 寅·卯·辰·巳·午·未·申·酉·戌·亥·子·丑
- 각 월령: `환경 3 / 요구 능력 3 / 관계 2 / 직업 2 / 재물 1 / 리스크 1`

# 01. 寅月 인월

**계절 자리:** 초봄 · 겨울의 저장이 풀리고 새 활동이 시작되는 때  
**지장간:** 戊·丙·甲  
**한 줄 핵심:** 인월은 멈춰 있던 환경에서 먼저 방향을 정하고 사람과 자원을 움직여 새 판을 시작해야 하는 월령이다.

## 월령 환경

- 아직 추위가 남아 있지만 성장과 경쟁이 동시에 시작된다.
- 기존 체계를 유지하는 일보다 새 목표와 선점이 중요해진다.
- 능력이 완성되기 전에도 먼저 움직여 자리를 잡아야 한다.

## 당령의 요구

- 시작할 일과 미룰 일을 빠르게 구분한다.
- 사람에게 방향과 초기 역할을 제시한다.
- 계획에 예산, 일정, 마감 기준을 붙인다.

## 사령 후보의 분화

- 戊: 기존 자원과 생활 기반을 점검하고 출발 조건을 마련한다.
- 丙: 얼어 있던 환경을 데우고 계획을 밖으로 드러낸다.
- 甲: 큰 방향을 세우고 성장과 개척을 주도한다.

## 조화·조후

- 조화: 추진력에 실무자, 예산, 마감이 붙어야 시작이 실제 성과로 남는다.
- 조후: 남은 냉기를 풀 화기가 필요하다. 수기가 지나치면 생각과 준비가 길어지고, 금기가 너무 이르면 시작부터 꺾이기 쉽다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 환경 | 새 판의 시작 |
| 환경 | 선점 경쟁 |
| 환경 | 빠른 역할 형성 |
| 요구 능력 | 초기 기획 |
| 요구 능력 | 사람 동원 |
| 요구 능력 | 실행 기준 설정 |
| 관계 | 주도권 관계 |
| 관계 | 성장 없는 관계에 싫증 |
| 직업 | 창업·신규 사업 |
| 직업 | 교육·영업 기획 |
| 재물 | 종잣돈·초기 투자 |
| 리스크 | 무리한 확장과 마감 부족 |

# 02. 卯月 묘월

**계절 자리:** 중춘 · 목이 가장 부드럽고 왕성하게 번지는 때  
**지장간:** 乙  
**한 줄 핵심:** 묘월은 사람, 취향, 관계의 반응을 세밀하게 읽고 성장한 것을 시장과 공동체 안에 넓혀야 하는 월령이다.

## 월령 환경

- 혼자 앞서기보다 사람 사이의 호감과 균형이 중요해진다.
- 외형, 말투, 품질, 분위기가 선택에 직접 영향을 준다.
- 관계가 넓어지는 만큼 비교와 평판 관리도 필요하다.

## 당령의 요구

- 사람과 고객의 반응을 읽고 표현 방식을 조정한다.
- 관계를 넓히되 가격과 거절 기준을 분명히 한다.
- 취향과 미감을 실제 결과물로 만든다.

## 사령 후보의 분화

- 乙: 관계를 부드럽게 잇고 상황에 맞춰 성장 경로를 바꾼다.

## 조화·조후

- 조화: 미감과 관계성이 표현, 상품, 교육, 서비스로 이어져야 한다.
- 조후: 화기가 있으면 성장한 것이 드러나고, 금기가 적절하면 품질과 기준이 생긴다. 수기가 지나치면 관계와 감정이 번지기만 쉽다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 환경 | 관계의 번성 |
| 환경 | 취향 경쟁 |
| 환경 | 평판 민감 |
| 요구 능력 | 관계 조율 |
| 요구 능력 | 미감의 상품화 |
| 요구 능력 | 가격·거절 기준 |
| 관계 | 호감과 비교 |
| 관계 | 부드러운 주도권 |
| 직업 | 디자인·브랜딩 |
| 직업 | 상담·교육·서비스 |
| 재물 | 고객 반응 수익 |
| 리스크 | 사람에게 맞추다 기준 상실 |

# 03. 辰月 진월

**계절 자리:** 늦봄 · 성장한 것을 거두어 다음 계절로 넘기는 습토의 때  
**지장간:** 乙·癸·戊  
**한 줄 핵심:** 진월은 성장, 정보, 자산이 뒤섞인 환경에서 무엇을 남기고 현실 기반으로 바꿀지 결정해야 하는 월령이다.

## 월령 환경

- 봄에 벌인 일의 가능성과 비용을 함께 계산해야 한다.
- 가족, 돈, 감정, 문서처럼 서로 다른 문제가 한데 얽히기 쉽다.
- 바로 결론내기보다 숨은 자원과 묵은 문제를 확인하는 일이 먼저다.

## 당령의 요구

- 복잡한 자료와 이해관계를 분류한다.
- 잠재 자원을 계약, 자산, 프로젝트로 전환한다.
- 늦어진 결정과 정산에 마감 시점을 정한다.

## 사령 후보의 분화

- 乙: 이어갈 성장과 관계를 추려 다음 단계로 넘긴다.
- 癸: 숨은 정보, 감정, 현금의 이동을 파악한다.
- 戊: 자원과 책임의 위치를 정하고 현실 기반을 만든다.

## 조화·조후

- 조화: 여러 가능성 가운데 우선순위를 정하고 계약·예산·담당자를 분명히 해야 한다.
- 조후: 화기가 습기를 걷고 내용을 드러내며, 금기가 형태와 기준을 만든다. 수토가 지나치면 계획과 돈이 오래 묶인다.

## 12개 핵심어

| 분류 | 핵심어 |
|---|---|
| 환경 | 복합 이해관계 |
| 환경 | 잠재 자원 |
| 환경 | 늦봄의 전환 |
| 요구 능력 | 자료 분류 |
| 요구 능력 | 자산 전환 |
| 요구 능력 | 결정·정산 마감 |
| 관계 | 감정과 실리의 결합 |
| 관계 | 가족·돈 문제의 얽힘 |
| 직업 | 부동산·자산 기획 |
| 직업 | 데이터·프로젝트 관리 |
| 재물 | 숨은 자산·묶인 돈 |
| 리스크 | 결정 지연과 묵은 문제 |

# 04. 巳月 사월

**계절 자리:** 초여름 · 열과 속도가 올라오고 기술이 단련되기 시작하는 때  
**지장간:** 戊·庚·丙  
**한 줄 핵심:** 사월은 시장과 사람의 반응을 빠르게 읽고 기술, 영업, 연출을 실제 성과로 보여줘야 하는 월령이다.

## 월령 환경

- 경쟁의 속도가 빨라지고 즉각적인 결과를 요구받는다.
- 기술과 이미지, 사람과 돈을 동시에 다루는 일이 많아진다.
- 정보가 늦으면 기회를 놓치지만 검증을 생략하면 사고가 커진다.

## 당령의 요구

- 기술과 도구를 익혀 짧은 시간 안에 결과를 낸다.
- 경쟁자, 고객, 거래 조건을 빠르게 파악한다.
- 과열된 상황에서도 계약과 품질 기준을 유지한다.

## 사령 후보의 분화

- 戊: 빠른 활동을 받쳐줄 운영 기반과 책임 범위를 정한다.
- 庚: 기술, 장비, 거래 기준으로 실제 문제를 해결한다.
- 丙: 성과를 밖으로 드러내고 사람과 시장을 움직인다.

## 조화·조후

- 조화: 속도와 욕망을 기술, 계약, 납기, 품질로 통제해야 한다.
- 조후: 수기가 열을 식히고 판단 시간을 확보한다. 토가 과하면 열이 갇히고, 화기가 지나치면 말과 행동이 앞서기 쉽다.

| 환경 | 기술 시장 |
| 환경 | 즉각적 성과 요구 |
| 요구 능력 | 기술 습득 |
| 요구 능력 | 거래 판단 |

# 05. 午月 오월

**계절 자리:** 한여름 · 화가 극성에 이르고 모든 것이 공개되는 때  
**지장간:** 己·丁  
**한 줄 핵심:** 오월은 사람들 앞에서 역할과 입장을 분명히 하고 평판, 설득, 존재감으로 결과를 만들어야 하는 월령이다.

## 월령 환경

- 성과와 감정이 숨겨지기 어렵고 대외 반응이 빠르다.
- 직책, 명예, 발표, 인기처럼 공개된 평가가 중요해진다.
- 사람을 끌어올리는 능력과 자기 감정을 다루는 능력이 함께 필요하다.

## 당령의 요구

- 공개된 자리에서 분명하게 말하고 결정한다.
- 조직과 사람의 분위기를 끌어올린다.
- 감정과 체면을 성과·책임·정산으로 연결한다.

## 사령 후보의 분화

- 己: 밖으로 펼친 활동을 생활 운영과 실제 결과로 정착시킨다.
- 丁: 사람의 시선을 모으고 집중된 표현과 영향력을 만든다.

## 조화·조후

- 조화: 표현력과 대중성에 책임, 실적, 사후 관리가 붙어야 한다.
- 조후: 수기가 과열을 식히고 금기가 결과를 평가한다. 화기가 지나치면 판단이 감정과 체면에 끌려간다.

# 06. 未月 미월

# 07. 申月 신월

**계절 자리:** 초가을 · 성장한 것을 선별하고 기술과 거래가 발동하는 때
