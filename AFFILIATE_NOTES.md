# 쿠팡 파트너스 적용 메모

이번 MVP의 광고 수익화는 쿠팡 파트너스를 기준으로 한다.

## 적용 원칙

- 운세 결과의 핵심 판단 위에는 광고를 두지 않는다.
- 광고는 결과를 모두 본 뒤 자연스럽게 이어지는 보조 영역에 둔다.
- 제휴 링크와 배너가 보이는 영역에는 대가성 고지 문구를 함께 둔다.
- 광고 문구는 운세 결과처럼 보이면 안 된다.
- 제휴 상품은 명리, 독서, 기록, 생활 관리처럼 서비스 맥락과 맞는 상품을 우선한다.

## 기본 고지 문구

```text
이 페이지는 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받을 수 있습니다.
```

## 1차 배치 후보

1. 프리미엄 결과 하단
2. 명식표 화면 하단
3. 이현 블로그 연결 버튼 주변
4. 추후 별도 콘텐츠 페이지

## 전환 팝업 방향

광고를 운세 이용 동선 안에 넣고, 분석 결과가 생성될 때마다 쿠팡 방문 팝업을 띄운다.

우리 서비스의 팝업은 다음 조건을 따른다.

- 사주 분석 결과가 생성된 직후 매번 띄운다.
- 화면을 어둡게 눌러 중앙 팝업으로 명확히 노출한다.
- 닫기 버튼과 계속 보기 버튼을 명확히 둔다.
- 쿠팡 이동 버튼은 광고임을 숨기지 않는다.
- 팝업 안에 대가성 고지 문구를 함께 둔다.

## 팝업 문구 초안

제목:

```text
쿠팡 방문하기
```

본문:

```text
제휴 페이지를 확인한 뒤 운세 결과를 계속 볼 수 있습니다.
```

고지:

```text
파트너스 활동으로 수수료를 받을 수 있습니다.
```

버튼:

```text
쿠팡 방문하고 결과 보기
결과 계속 보기
```

## 팝업 노출 후보

1. 프리미엄 분석 결과 생성 직후
2. 사용자가 같은 사주를 다시 분석한 경우에도 재노출
3. 실제 파트너스 링크가 준비되면 `window.LEEHYEON_COUPANG_PARTNERS_URL` 값으로 교체
4. 실제 상품 이미지가 준비되면 `window.LEEHYEON_COUPANG_POPUP_IMAGE_URL` 값으로 교체

## 팝업에서 피해야 할 표현

- 운세가 좋아지려면 구매해야 한다는 식의 암시
- 불안감을 자극해 상품 구매를 유도하는 문장
- 쿠팡 이동을 결과 확인의 필수 조건처럼 보이게 하는 방식
- 닫기 버튼을 작게 만들거나 숨기는 방식
- 광고와 운세 분석을 같은 카드 안에 섞는 방식

## 피해야 할 배치

- 생년월일 입력 직전
- 핵심 운세 지표 사이
- 성격, 재물, 직업 등 주요 결과 카드 내부
- 사용자가 광고와 분석 결과를 혼동할 수 있는 위치
